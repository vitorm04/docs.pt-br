# <a name="icordebugcodegetaddress-method"></a>Método ICorDebugCode::GetAddress
Obtém o endereço virtual relativo (RVA) do segmento de código que representa essa interface "ICorDebugCode".  
  
## <a name="syntax"></a>Sintaxe  
  
```  
HRESULT GetAddress (  
    [out] CORDB_ADDRESS *pStart  
);  
```  
  
#### <a name="parameters"></a>Parâmetros  
 `pStart`  
 [out] Um ponteiro para o RVA do segmento de código.
