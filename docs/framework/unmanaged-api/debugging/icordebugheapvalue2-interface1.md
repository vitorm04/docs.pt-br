# <a name="icordebugheapvalue2-interface"></a>Interface ICorDebugHeapValue2

Uma extensão de ICorDebugHeapValue que fornece suporte para identificadores de Common Language Runtime (CLR).  
  
## <a name="methods"></a>Métodos  
  
|Método|Descrição|  
|------------|-----------------|  
|[Método CreateHandle](../../../../docs/framework/unmanaged-api/debugging/icordebugheapvalue2-createhandle-method.md)|Cria um identificador do tipo especificado para esse `ICorDebugHeapValue2` objeto.|  
  
## <a name="remarks"></a>Comentários  
  
> [!NOTE]
> Esta interface não dá suporte para chamada remota, seja entre computadores ou processos cruzados.
