# <a name="corlinkeroptions-enumeration"></a>Enumeração CorLinkerOptions
Especifica os sinalizadores para selecionar opções para o vinculador de metadados.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
typedef enum CorLinkerOptions {  
    MDAssembly          = 0x00000000,  
    MDNetModule         = 0x00000001,  
} CorLinkerOptions;  
```  
  
## <a name="members"></a>Membros  
  
|Membro|Descrição|  
|------------|-----------------|  
|`MDAssembly`|As funções globais e tipos privados não são preservadas.|  
|`MDNetModule`|As funções globais e tipos privados são preservadas.|
