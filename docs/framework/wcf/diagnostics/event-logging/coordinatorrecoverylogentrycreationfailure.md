# <a name="coordinatorrecoverylogentrycreationfailure"></a>CoordinatorRecoveryLogEntryCreationFailure
ID: 140  
  
 Severidade: erro  
  
 Categoria: TransactionBridge  
  
## <a name="description"></a>Descrição  
 Uma entrada de log de recuperação não pôde ser gerada para a inscrição de coordenador. A transação será anulada. Listas de eventos a ID da transação, motivo, exceção, o nome do processo e processo de identificação.
