# <a name="systemservicemodelchannelspeerneighbormanageronline"></a>System.ServiceModel.Channels.PeerNeighborManagerOnline
O PeerNode agora está online, que indica a presença de um ou mais vizinhos conectados.  
  
## <a name="description"></a>Descrição  
 Este rastreamento ocorre na conclusão de uma tentativa de conexão do vizinho com êxito.
