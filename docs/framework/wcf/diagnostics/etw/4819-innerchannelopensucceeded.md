# <a name="4819---innerchannelopensucceeded"></a>4819 - InnerChannelOpenSucceeded
## <a name="properties"></a>Propriedades  
  
|||  
|-|-|  
|ID|4819|  
|Palavras-chave|Descoberta|  
|Nível|Informações|  
|Canal|Os aplicativos de servidor de Microsoft-Windows- aplicativo/depuração|  
  
## <a name="description"></a>Descrição  
 Esse evento é emitido quando o DiscoveryClientChannel descobriu um ponto de extremidade com êxito e abriu o canal usá-lo.  
  
## <a name="message"></a>Mensagem  
 O DiscoveryClientChannel descobriu um ponto de extremidade com êxito e abriu o canal usá-lo. O cliente está conectado a um serviço usando EndpointAddress = '%1' e Via = '%2'.
