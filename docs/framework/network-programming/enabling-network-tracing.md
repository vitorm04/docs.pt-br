 <span data-ttu-id="58b42-110">Os destinos de rastreamento mais comuns, também conhecidos como ouvintes de rastreamento, são o ouvinte padrão e o arquivo de log.</span><span class="sxs-lookup"><span data-stu-id="58b42-110">The most common trace destinations, also referred to as trace listeners, are the default listener and the log file.</span></span>  
  
 <span data-ttu-id="58b42-111">O rastreamento usará o ouvinte padrão se você não especificar um ouvinte de rastreamento.</span><span class="sxs-lookup"><span data-stu-id="58b42-111">Tracing uses the default listener if you do not specify a trace listener.</span></span> <span data-ttu-id="58b42-112">Exiba as mensagens enviadas para o ouvinte padrão executando o código em um depurador habilitado para código gerenciado, como o Depurador CLR fornecido com o SDK do .NET Framework ou o DBwin32.exe fornecido com o SDK do Windows.</span><span class="sxs-lookup"><span data-stu-id="58b42-112">You can view messages sent to the default listener by executing your code in a managed code-enabled debugger such as the CLR Debugger shipped with the .NET Framework SDK, or DBwin32.exe shipped with the Windows SDK.</span></span> <span data-ttu-id="58b42-113">Usando o Depurador CLR, as mensagens de rastreamento são exibidas na janela **Saída**.</span><span class="sxs-lookup"><span data-stu-id="58b42-113">Using the CLR Debugger, the trace messages appear in the **Output** window.</span></span>  
  
 <span data-ttu-id="58b42-114">Se preferir usar um arquivo para receber rastreamentos, especifique um arquivo de log usando as definições de configuração, conforme mostrado no exemplo a seguir.</span><span class="sxs-lookup"><span data-stu-id="58b42-114">If you prefer to use a file to receive traces, you can specify a log file by using configuration settings, as shown in the following example.</span></span> <span data-ttu-id="58b42-115">(Para obter uma discussão geral sobre arquivos de configuração, consulte [Arquivos de configuração](../configure-apps/index.md).)</span><span class="sxs-lookup"><span data-stu-id="58b42-115">(For a general discussion of configuration files, see [Configuration Files](../configure-apps/index.md).)</span></span>  
  
 <span data-ttu-id="58b42-116">Para enviar os rastreamentos para um arquivo de log, adicione o nó a seguir ao nó `<system.diagnostics>` do arquivo de configuração apropriado (aplicativo ou computador).</span><span class="sxs-lookup"><span data-stu-id="58b42-116">To send traces to a log file, add the following node to the `<system.diagnostics>` node of the appropriate configuration file (application or machine).</span></span> <span data-ttu-id="58b42-117">Altere o nome do arquivo (trace.log) de acordo com suas necessidades.</span><span class="sxs-lookup"><span data-stu-id="58b42-117">You can change the name of the file (trace.log) to suit your needs.</span></span>  
  
```xml  
<system.diagnostics>  
  <trace autoflush="true" indentsize="4">  
    <listeners>  
      <add name="file" type="System.Diagnostics.TextWriterTraceListener" initializeData="trace.log"/>  
    </listeners>   
  </trace>  
</system.diagnostics>  
```
