# <a name="socket-code-examples"></a>Exemplos de código de soquete
Os exemplos de código a seguir demonstram como usar a classe <xref:System.Net.Sockets.Socket> como um cliente para se conectar aos serviços de rede remota e como um servidor para escutar conexões de clientes remotos.  
  
## <a name="in-this-section"></a>Nesta seção  
 [Exemplo de soquete de cliente síncrono](../../../docs/framework/network-programming/synchronous-client-socket-example.md)  
 Mostra como implementar um cliente <xref:System.Net.Sockets.Socket> síncrono que se conecta a um servidor e exibe os dados retornados do servidor.  
  
 [Exemplo de soquete de servidor síncrono](../../../docs/framework/network-programming/synchronous-server-socket-example.md)  
 Mostra como implementar um servidor <xref:System.Net.Sockets.Socket> síncrono que aceita conexões de um cliente e retorna os dados recebidos do cliente.  
  
 [Exemplo de soquete de cliente assíncrono](../../../docs/framework/network-programming/asynchronous-client-socket-example.md)  
 Mostra como implementar um cliente <xref:System.Net.Sockets.Socket> assíncrono que se conecta a um servidor e exibe os dados retornados do servidor.  
  
 [Exemplo de soquete de servidor assíncrono](../../../docs/framework/network-programming/asynchronous-server-socket-example.md)  
 Mostra como implementar um servidor <xref:System.Net.Sockets.Socket> assíncrono que aceita conexões de um cliente e retorna os dados recebidos do cliente.
