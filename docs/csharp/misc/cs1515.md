# <a name="compiler-error-cs1515"></a>Erro do Compilador CS1515
'in' esperado  
  
 Em um [foreach, em](../language-reference/keywords/foreach-in.md) declaração, a parte "em" está faltando.  
  
## <a name="example"></a>Exemplo  
 A seguinte amostra gera CS1515:  
  
```csharp
using System;  
  
class Driver  
{  
   static void Main()  
   {  
      int[] arr = new int[] {1, 2, 3};  
  
      // try the following line instead  
      // foreach (int x in arr)  
      foreach (int x arr)      // CS1515, "in" is missing  
      {  
         Console.WriteLine(x);  
      }  
   }  
}  
```
