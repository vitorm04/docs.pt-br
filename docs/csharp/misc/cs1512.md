# <a name="compiler-error-cs1512"></a>CS1512 de erro do compilador
Palavra-chave 'base' não está disponível no contexto atual  
  
 O [base](../../csharp/language-reference/keywords/base.md) palavra-chave foi usado fora de um método, propriedade ou construtor.  
  
 O exemplo a seguir gera CS1512:  
  
```  
// CS1512.cs  
using System;  
  
class Base {}  
  
class CMyClass : Base  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
