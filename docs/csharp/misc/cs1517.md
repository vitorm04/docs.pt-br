# <a name="compiler-error-cs1517"></a>CS1517 de erro do compilador
Expressão do pré-processador inválido  
  
 O compilador encontrou uma expressão do pré-processador inválida.  
  
 Para obter mais informações, consulte [diretivas de pré-processador](../../csharp/language-reference/preprocessor-directives/index.md).  
  
 O exemplo a seguir mostra algumas expressões de pré-processador válidas e inválidas:  
  
```  
// CS1517.cs  
#if symbol      // OK  
#endif  
#if !symbol     // OK  
#endif  
#if (symbol)    // OK  
#endif  
#if true        // OK  
#endif  
#if false       // OK  
#endif  
#if 1           // CS1517  
#endif  
#if ~symbol     // CS1517  
#endif  
#if *           // CS1517  
#endif  
  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
