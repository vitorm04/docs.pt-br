# <a name="compiler-error-cs1623"></a><span data-ttu-id="d6ed6-102">CS1623 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="d6ed6-102">Compiler Error CS1623</span></span>
<span data-ttu-id="d6ed6-103">Iteradores não podem ter em ref ou out parâmetros</span><span class="sxs-lookup"><span data-stu-id="d6ed6-103">Iterators cannot have in ref or out parameters</span></span>  
  
 <span data-ttu-id="d6ed6-104">Esse erro ocorre se um método iterador usa um `in`, `ref`, ou `out` parâmetro.</span><span class="sxs-lookup"><span data-stu-id="d6ed6-104">This error occurs if an iterator method takes an `in`, `ref`, or `out` parameter.</span></span> <span data-ttu-id="d6ed6-105">Para evitar esse erro, remova o `in`, `ref`, ou `out` palavra-chave de assinatura de método.</span><span class="sxs-lookup"><span data-stu-id="d6ed6-105">To avoid this error, remove the `in`, `ref`, or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d6ed6-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="d6ed6-106">Example</span></span>  
 <span data-ttu-id="d6ed6-107">O exemplo a seguir gera CS1623:</span><span class="sxs-lookup"><span data-stu-id="d6ed6-107">The following sample generates CS1623:</span></span>  
  
```csharp  
// CS1623.cs  
using System.Collections;

class C : IEnumerable
{
    public IEnumerator GetEnumerator()
    {
        yield return 0;
    }

    // To resolve the error, remove in  
    public IEnumerator GetEnumerator(in short i)  // CS1623  
    {
        yield return i;
    }
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {
        yield return i;
    }

    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {
        f = 0.0F;
        yield return f;
    }
}
```
