# <a name="compiler-error-cs1575"></a><span data-ttu-id="b8df2-102">Erro do compilador CS1575</span><span class="sxs-lookup"><span data-stu-id="b8df2-102">Compiler Error CS1575</span></span>
<span data-ttu-id="b8df2-103">Uma expressão stackalloc requer [] após o tipo</span><span class="sxs-lookup"><span data-stu-id="b8df2-103">A stackalloc expression requires [] after type</span></span>  
  
 <span data-ttu-id="b8df2-104">O tamanho da alocação solicitada, com [stackalloc](../language-reference/operators/stackalloc.md), deve ser especificado entre colchetes.</span><span class="sxs-lookup"><span data-stu-id="b8df2-104">The size of the requested allocation, with [stackalloc](../language-reference/operators/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="b8df2-105">O exemplo a seguir gera CS1575:</span><span class="sxs-lookup"><span data-stu-id="b8df2-105">The following sample generates CS1575:</span></span>  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
