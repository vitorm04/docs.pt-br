# <a name="compiler-error-cs1101"></a><span data-ttu-id="f9f61-102">CS1101 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="f9f61-102">Compiler Error CS1101</span></span>
<span data-ttu-id="f9f61-103">O modificador de parâmetro 'ref' não pode ser usado com 'this'.</span><span class="sxs-lookup"><span data-stu-id="f9f61-103">The parameter modifier 'ref' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="f9f61-104">Quando o `this` palavra-chave modifica o primeiro parâmetro de um método estático, ele informa ao compilador que o método é um método de extensão.</span><span class="sxs-lookup"><span data-stu-id="f9f61-104">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="f9f61-105">Outros modificadores não são necessários ou permitidos no primeiro parâmetro de um método de extensão.</span><span class="sxs-lookup"><span data-stu-id="f9f61-105">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f9f61-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="f9f61-106">Example</span></span>  
 <span data-ttu-id="f9f61-107">O exemplo a seguir gera CS1101:</span><span class="sxs-lookup"><span data-stu-id="f9f61-107">The following example generates CS1101:</span></span>  
  
```csharp  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```
