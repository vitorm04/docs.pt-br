 <span data-ttu-id="ce594-104">Um **adicionar** ou **remover** palavra-chave em um [evento](../../csharp/language-reference/keywords/event.md) definição deve ter um corpo.</span><span class="sxs-lookup"><span data-stu-id="ce594-104">An **add** or **remove** keyword in an [event](../../csharp/language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="ce594-105">Para obter mais informações, consulte [Eventos](../../csharp/programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="ce594-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="ce594-106">O exemplo a seguir gera CS0073:</span><span class="sxs-lookup"><span data-stu-id="ce594-106">The following sample generates CS0073:</span></span>  
  
```csharp  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
