# <a name="compiler-error-cs1011"></a>Erro do compilador CS1011
Literal de caractere vazio  
  
 Um [char](../../csharp/language-reference/keywords/char.md) foi declarada e inicializada com um valor nulo. A inicialização de um `char` deve especificar um caractere.  
  
 O exemplo a seguir gera CS1011:  
  
```csharp  
// CS1011.cs  
class Sample  
{  
   public char CharField = '';   // CS1011  
}  
```
