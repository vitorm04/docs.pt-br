 <span data-ttu-id="b203e-105">A especificação da linguagem C# não permite código não seguro em iteradores.</span><span class="sxs-lookup"><span data-stu-id="b203e-105">The C# language specification does not allow unsafe code in iterators.</span></span>  
  
 <span data-ttu-id="b203e-106">O exemplo a seguir gera CS1629:</span><span class="sxs-lookup"><span data-stu-id="b203e-106">The following sample generates CS1629:</span></span>

```csharp  
class C
{  
   IEnumerator<int> IteratorMeth() {  
      int i;

         int *p = &i;

      }  
   }  
}  
```
