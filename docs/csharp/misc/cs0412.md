## <a name="example"></a><span data-ttu-id="d66e3-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="d66e3-106">Example</span></span>  
 <span data-ttu-id="d66e3-107">O exemplo a seguir gera CS0412:</span><span class="sxs-lookup"><span data-stu-id="d66e3-107">The following sample generates CS0412:</span></span>  
  
```  
// CS0412.cs  
using System;  
  
class C  
{  
    // Parameter name is the same as method type parameter name  
    public void G<T>(int T)  // CS0412  
    {  
    }  
    public void F<T>()  
    {  
        // Method local variable name is the same as method type  
        // parameter name  
        double T = 0.0;  // CS0412  
        Console.WriteLine(T);  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
