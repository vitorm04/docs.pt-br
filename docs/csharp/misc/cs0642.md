 <span data-ttu-id="5a62a-106">O exemplo a seguir gera CS0642:</span><span class="sxs-lookup"><span data-stu-id="5a62a-106">The following sample generates CS0642:</span></span>  
  
```  
// CS0642.cs  
// compile with: /W:3  
class MyClass  
{  
   public static void Main()  
   {  
      int i;  
  
      for (i = 0; i < 10; i += 1);   // CS0642 semicolon intentional?  
      {  
         System.Console.WriteLine (i);  
      }  
   }  
}  
```
