# <a name="compiler-error-cs0569"></a>Erro do compilador CS0569
'method2': não é possível substituir 'method1' porque ele não é suportado pelo idioma  
  
 Esse erro ocorre quando você derivar de uma classe base que foi escrita em outra linguagem e o compilador não oferece suporte ao método que você está tentando substituir.
