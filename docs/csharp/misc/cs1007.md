<span data-ttu-id="8e076-104">Acessador de propriedade já definido</span><span class="sxs-lookup"><span data-stu-id="8e076-104">Property accessor already defined</span></span>  
  
 <span data-ttu-id="8e076-105">Ao declarar uma [Propriedade](../programming-guide/classes-and-structs/using-properties.md), você também deve declarar seus métodos de acessador.</span><span class="sxs-lookup"><span data-stu-id="8e076-105">When you declare a [property](../programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="8e076-106">No entanto, uma propriedade não pode ter mais de um `get` método de acessador ou mais de um `set` método de acessador.</span><span class="sxs-lookup"><span data-stu-id="8e076-106">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8e076-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="8e076-107">Example</span></span>  

 <span data-ttu-id="8e076-108">O exemplo a seguir gera CS1007:</span><span class="sxs-lookup"><span data-stu-id="8e076-108">The following sample generates CS1007:</span></span>

```csharp  
    public int MyProperty  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method

            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
