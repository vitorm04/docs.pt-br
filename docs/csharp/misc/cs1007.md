## <a name="example"></a><span data-ttu-id="53f4c-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="53f4c-106">Example</span></span>

```csharp  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method

        }  
    }  
  
    public static void Main() {}  
}  
```
