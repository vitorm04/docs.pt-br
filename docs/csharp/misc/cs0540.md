## <a name="example"></a><span data-ttu-id="7a6d5-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="7a6d5-107">Example</span></span>  
 <span data-ttu-id="7a6d5-108">O exemplo a seguir gera CS0540.</span><span class="sxs-lookup"><span data-stu-id="7a6d5-108">The following sample generates CS0540.</span></span>  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```
