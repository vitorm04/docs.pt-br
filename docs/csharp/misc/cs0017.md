 <span data-ttu-id="5fd46-105">Um programa só pode ter um método [Main](../programming-guide/main-and-command-args/index.md) .</span><span class="sxs-lookup"><span data-stu-id="5fd46-105">A program can only have one [Main](../programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="5fd46-106">Para resolver esse erro, você pode excluir todos os métodos principais em seu código, exceto um, ou pode usar a opção [-Main](../language-reference/compiler-options/main-compiler-option.md) do compilador para especificar qual método principal você deseja usar.</span><span class="sxs-lookup"><span data-stu-id="5fd46-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="5fd46-107">O exemplo a seguir gera CS0017:</span><span class="sxs-lookup"><span data-stu-id="5fd46-107">The following sample generates CS0017:</span></span>  
  
```csharp

{  
   static public void Main()  
   {  
   }  
}

{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
