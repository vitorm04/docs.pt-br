# <a name="compiler-error-cs0550"></a>CS0550 de erro do compilador
'acessador' adiciona um assessor não encontrado no membro de interface 'property'  
  
 A implementação de uma propriedade em uma classe derivada contém um acessador que não foi especificado na interface de base.  
  
 Para obter mais informações, consulte [Propriedades using](../../csharp/programming-guide/classes-and-structs/using-properties.md).  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera CS0550.  
  
```  
// CS0550.cs  
namespace x  
{  
   interface ii  
   {  
      int i  
      {  
         get;  
         // add the following accessor to resolve this CS0550  
         // set;  
      }  
   }  
  
   public class a : ii  
   {  
      int ii.i  
      {  
         get  
         {  
            return 0;  
         }  
         set {}   // CS0550  no set in interface  
      }  
  
      public static void Main() {}  
   }  
}  
```
