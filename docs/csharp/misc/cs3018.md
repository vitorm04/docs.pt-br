## <a name="example"></a><span data-ttu-id="d2a72-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="d2a72-108">Example</span></span>  

 <span data-ttu-id="d2a72-109">O exemplo a seguir gera CS3018.</span><span class="sxs-lookup"><span data-stu-id="d2a72-109">The following sample generates CS3018.</span></span>  
  
```csharp  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
