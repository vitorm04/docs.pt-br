# <a name="compiler-error-cs0669"></a><span data-ttu-id="e95a1-103">Erro do Compilador CS0669</span><span class="sxs-lookup"><span data-stu-id="e95a1-103">Compiler Error CS0669</span></span>

<span data-ttu-id="e95a1-104">Uma classe com o atributo ComImport não pode ter um construtor definido pelo usuário.</span><span class="sxs-lookup"><span data-stu-id="e95a1-104">A class with the ComImport attribute cannot have a user-defined constructor.</span></span>

<span data-ttu-id="e95a1-105">A camada de interoperabilidade COM no Common Language Runtime fornece o construtor para classes [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) .</span><span class="sxs-lookup"><span data-stu-id="e95a1-105">The COM interop layer in the common language runtime supplies the constructor for [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) classes.</span></span> <span data-ttu-id="e95a1-106">Consequentemente, um objeto COM pode ser usado como um objeto gerenciado no tempo de execução.</span><span class="sxs-lookup"><span data-stu-id="e95a1-106">Consequently, a COM object can be used as a managed object in the runtime.</span></span>

<span data-ttu-id="e95a1-107">O exemplo a seguir gera CS0669:</span><span class="sxs-lookup"><span data-stu-id="e95a1-107">The following sample generates CS0669:</span></span>

```csharp
// CS0669.cs
using System.Runtime.InteropServices;
[ComImport, Guid("00000000-0000-0000-0000-000000000001")]
class TestClass
{
   TestClass()   // CS0669, delete constructor to resolve
   {
   }

   public static void Main()
   {
   }
}
```
