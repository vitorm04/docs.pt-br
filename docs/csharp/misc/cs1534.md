# <a name="compiler-error-cs1534"></a>Erro do compilador CS1534
Operador binário sobrecarregado 'operator' utiliza dois parâmetros  
  
 A definição de um binário [operador](../../csharp/language-reference/keywords/operator.md) deve levar dois parâmetros.  
  
 O exemplo a seguir gera CS1534:  
  
```csharp  
// CS1534.cs  
class MyClass  
{  
   public static MyClass operator - (MyClass MC1, MyClass MC2, MyClass MC3)   // CS1534  
   // try the following line instead  
   // public static MyClass operator - (MyClass MC1, MyClass MC2)  
   {  
      return new MyClass();  
   }  
  
   public static int Main()  
   {  
      return 1;  
   }  
}  
```
