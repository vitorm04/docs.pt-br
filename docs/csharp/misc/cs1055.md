# <a name="compiler-error-cs1055"></a><span data-ttu-id="31ecb-103">Erro do Compilador CS1055</span><span class="sxs-lookup"><span data-stu-id="31ecb-103">Compiler Error CS1055</span></span>

<span data-ttu-id="31ecb-104">Um acessador Add ou remove esperado</span><span class="sxs-lookup"><span data-stu-id="31ecb-104">An add or remove accessor expected</span></span>  
  
 <span data-ttu-id="31ecb-105">Se o [evento](../language-reference/keywords/event.md) não for declarado como um campo, ele deverá definir as funções de acessador **Add** e **Remove** .</span><span class="sxs-lookup"><span data-stu-id="31ecb-105">If your [event](../language-reference/keywords/event.md) is not declared as a field, it must define both **add** and **remove** accessor functions.</span></span>  
  
 <span data-ttu-id="31ecb-106">O exemplo a seguir gera CS1055:</span><span class="sxs-lookup"><span data-stu-id="31ecb-106">The following sample generates CS1055:</span></span>  
  
```csharp  
// CS1055.cs  
delegate void del();  
class Test  
{  
   public event del MyEvent  
   {  
      int i;   // CS1055  
      // uncomment accessors and delete previous line to resolve  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      // remove  
      // {  
      //    MyEvent -= value;  
      // }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
