# <a name="compiler-error-cs1055"></a><span data-ttu-id="e74e2-102">Erro do compilador CS1055</span><span class="sxs-lookup"><span data-stu-id="e74e2-102">Compiler Error CS1055</span></span>
<span data-ttu-id="e74e2-103">Um acessador Add ou remove esperado</span><span class="sxs-lookup"><span data-stu-id="e74e2-103">An add or remove accessor expected</span></span>  
  
 <span data-ttu-id="e74e2-104">Se o [evento](../language-reference/keywords/event.md) não for declarado como um campo, ele deverá definir as funções de acessador **Add** e **Remove** .</span><span class="sxs-lookup"><span data-stu-id="e74e2-104">If your [event](../language-reference/keywords/event.md) is not declared as a field, it must define both **add** and **remove** accessor functions.</span></span>  
  
 <span data-ttu-id="e74e2-105">O exemplo a seguir gera CS1055:</span><span class="sxs-lookup"><span data-stu-id="e74e2-105">The following sample generates CS1055:</span></span>  
  
```csharp  
// CS1055.cs  
delegate void del();  
class Test  
{  
   public event del MyEvent  
   {  
      int i;   // CS1055  
      // uncomment accessors and delete previous line to resolve  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      // remove  
      // {  
      //    MyEvent -= value;  
      // }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
