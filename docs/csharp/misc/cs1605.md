# <a name="compiler-error-cs1605"></a>Erro do compilador CS1605
Não é possível passar 'var' como ref ou out argumento porque ele é somente leitura  
  
 Uma variável passada como um [ref](../../csharp/language-reference/keywords/ref.md) ou [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parâmetro deve ser modificado no método chamado. Portanto, não é possível passar um parâmetro somente leitura, como `ref` ou `out`.
