 <span data-ttu-id="da062-104">Um [readonly](../../csharp/language-reference/keywords/readonly.md) variável deve ter o mesmo [estático](../../csharp/language-reference/keywords/static.md) uso do construtor no qual você deseja para inicializá-lo.</span><span class="sxs-lookup"><span data-stu-id="da062-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) variable must have the same [static](../../csharp/language-reference/keywords/static.md) usage as the constructor in which you want to initialize it.</span></span> <span data-ttu-id="da062-105">Para obter mais informações, consulte [Construtores estáticos](../../csharp/programming-guide/classes-and-structs/static-constructors.md).</span><span class="sxs-lookup"><span data-stu-id="da062-105">For more information, see [Static Constructors](../../csharp/programming-guide/classes-and-structs/static-constructors.md).</span></span>
