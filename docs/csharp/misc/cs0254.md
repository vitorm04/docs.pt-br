# <a name="compiler-error-cs0254"></a><span data-ttu-id="0bf3a-103">Erro do Compilador CS0254</span><span class="sxs-lookup"><span data-stu-id="0bf3a-103">Compiler Error CS0254</span></span>

 <span data-ttu-id="0bf3a-105">O lado direito de uma expressão [fixa](../language-reference/keywords/fixed-statement.md) não pode usar uma conversão.</span><span class="sxs-lookup"><span data-stu-id="0bf3a-105">The right side of a [fixed](../language-reference/keywords/fixed-statement.md) expression may not use a cast.</span></span> <span data-ttu-id="0bf3a-106">Para obter mais informações, consulte [Código não seguro e ponteiros](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="0bf3a-106">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="0bf3a-107">O exemplo a seguir gera CS0254:</span><span class="sxs-lookup"><span data-stu-id="0bf3a-107">The following sample generates CS0254:</span></span>  
  
```csharp  
// CS0254.cs  
// compile with: /unsafe  
class Point  
{  
   public uint x, y;  
}  
  
class FixedTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      *p *= *p;  
   }  
  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
      pt.x = 5;  
      pt.y = 6;  
  
      fixed (int* p = (int*)&pt.x)   // CS0254  
      // try the following line instead  
      // fixed (uint* p = &pt.x)  
      {  
         SquarePtrParam ((int*)p);  
      }  
   }  
}  
```
