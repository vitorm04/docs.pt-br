 <span data-ttu-id="f0b40-106">O exemplo a seguir gera CS0253:</span><span class="sxs-lookup"><span data-stu-id="f0b40-106">The following sample generates CS0253:</span></span>  
  
```csharp 
// CS0253.cs

using System;  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool c = s == o;   // CS0253  
      // try the following line instead  
      // bool c = s == (string)o;  
   }  
}  
```
