# <a name="compiler-error-cs0724"></a>Erro do compilador CS0724
não precisa de um atributo CLSCompliant porque o assembly não tem um atributo CLSCompliant  
  
 O exemplo a seguir gera CS0724 devido a `throw` instrução dentro do `finally` bloco de cláusula.  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera CS0724.  
  
```csharp  
// CS0724.cs  
using System;  
  
class X  
{  
    static void Test()  
    {  
        try  
        {  
            throw new Exception();  
        }  
        catch  
        {  
            try  
            {  
            }  
            finally  
            {  
                throw; // CS0724  
            }  
        }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
