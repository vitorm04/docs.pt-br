# <a name="compiler-error-cs0637"></a><span data-ttu-id="cec39-102">CS0637 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="cec39-102">Compiler Error CS0637</span></span>

 <span data-ttu-id="cec39-104">O [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) atributo não pode ser usado em campos marcados com [estático](../../csharp/language-reference/keywords/static.md) ou [const](../../csharp/language-reference/keywords/const.md).</span><span class="sxs-lookup"><span data-stu-id="cec39-104">The [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute cannot be used on fields marked with [static](../../csharp/language-reference/keywords/static.md) or [const](../../csharp/language-reference/keywords/const.md).</span></span>  
  
 <span data-ttu-id="cec39-105">O exemplo a seguir gera CS0637:</span><span class="sxs-lookup"><span data-stu-id="cec39-105">The following sample generates CS0637:</span></span>  
  
```  
// CS0637.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
public class MainClass  
{  
   [FieldOffset(3)]   // CS0637  
   public static int i;  
   public static void Main ()  
   {  
   }  
}  
```
