# <a name="compiler-error-cs1618"></a><span data-ttu-id="bdfba-103">Erro do Compilador CS1618</span><span class="sxs-lookup"><span data-stu-id="bdfba-103">Compiler Error CS1618</span></span>

<span data-ttu-id="bdfba-104">Não é possível criar delegate com ' method ' porque ele tem um atributo Conditional</span><span class="sxs-lookup"><span data-stu-id="bdfba-104">Cannot create delegate with 'method' because it has a Conditional attribute</span></span>  
  
 <span data-ttu-id="bdfba-105">Você não pode criar um delegado com um método condicional porque o método pode não existir em algumas compilações.</span><span class="sxs-lookup"><span data-stu-id="bdfba-105">You cannot create a delegate with a conditional method because the method might not exist in some builds.</span></span>  
  
 <span data-ttu-id="bdfba-106">O exemplo a seguir gera CS1618:</span><span class="sxs-lookup"><span data-stu-id="bdfba-106">The following sample generates CS1618:</span></span>  
  
```csharp  
// CS1618.cs  
using System;  
using System.Diagnostics;  
  
delegate void del();  
  
class MakeAnError {  
   public static void Main() {  
      del d = new del(ConditionalMethod);   // CS1618  
      // Invalid because on builds where DEBUG is not set,
      // there will be no "ConditionalMethod".  
   }  
   // To fix the error, remove the next line:  
   [Conditional("DEBUG")]  
   public static void ConditionalMethod()
   {  
      Console.WriteLine("Do something only in debug");  
   }  
}  
```
