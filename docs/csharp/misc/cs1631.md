 <span data-ttu-id="b54be-104">A instrução yield não é permitida de dentro do corpo de uma cláusula catch.</span><span class="sxs-lookup"><span data-stu-id="b54be-104">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="b54be-105">Para evitar esse erro, mova a instrução yield fora do corpo de uma cláusula catch.</span><span class="sxs-lookup"><span data-stu-id="b54be-105">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>

```  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()

      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }    
  
   public static void Main()   
   {  
   }  
}  
```
