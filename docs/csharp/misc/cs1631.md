 <span data-ttu-id="a90e9-104">A instrução yield não é permitida de dentro do corpo de uma cláusula catch.</span><span class="sxs-lookup"><span data-stu-id="a90e9-104">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="a90e9-105">Para evitar esse erro, mova a instrução yield fora do corpo de uma cláusula catch.</span><span class="sxs-lookup"><span data-stu-id="a90e9-105">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="a90e9-106">O exemplo a seguir gera CS1631:</span><span class="sxs-lookup"><span data-stu-id="a90e9-106">The following sample generates CS1631:</span></span>  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()   
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }    
  
   public static void Main()   
   {  
   }  
}  
```
