# <a name="compiler-error-cs0584"></a>Erro do compilador CS0584
Erro do compilador interno: símbolo de "estágio" estágio 'symbol'  
  
 Tente determinar se o compilador está falhando em razão de sua incapacidade de analisar sintaxes inesperadas. Se não for o caso, tente [Fale conosco](/visualstudio/ide/talk-to-us).
