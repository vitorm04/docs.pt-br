# <a name="compiler-error-cs0538"></a><span data-ttu-id="14e5c-102">CS0538 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="14e5c-102">Compiler Error CS0538</span></span>
<span data-ttu-id="14e5c-103">'name' na declaração de interface explícita não é uma interface</span><span class="sxs-lookup"><span data-stu-id="14e5c-103">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="14e5c-104">Foi feita uma tentativa para declarar explicitamente um [interface](../../csharp/language-reference/keywords/interface.md), mas uma interface não foi especificada.</span><span class="sxs-lookup"><span data-stu-id="14e5c-104">An attempt was made to explicitly declare an [interface](../../csharp/language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="14e5c-105">O exemplo a seguir gera CS0538:</span><span class="sxs-lookup"><span data-stu-id="14e5c-105">The following sample generates CS0538:</span></span>  
  
```  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
