# <a name="compiler-error-cs5001"></a><span data-ttu-id="4de4d-102">CS5001 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="4de4d-102">Compiler Error CS5001</span></span>

 <span data-ttu-id="4de4d-104">Esse erro ocorre quando não há estático [principal](../../csharp/programming-guide/main-and-command-args/index.md) método com uma assinatura correta é encontrado no código que produz um arquivo executável.</span><span class="sxs-lookup"><span data-stu-id="4de4d-104">This error occurs when no static [Main](../../csharp/programming-guide/main-and-command-args/index.md) method with a correct signature is found in the code that produces an executable file.</span></span> <span data-ttu-id="4de4d-105">Esse erro também ocorrerá se a entrada do ponto de função, `Main`, é definida com o caso errado, como em letras minúsculas `main`.</span><span class="sxs-lookup"><span data-stu-id="4de4d-105">This error also occurs if the entry point function, `Main`, is defined with the wrong case, such as lower-case `main`.</span></span>  
  
-   <span data-ttu-id="4de4d-106">`Main` deve ser declarado como estático e ele devem retornar [void](../../csharp/language-reference/keywords/void.md) ou [int](../../csharp/language-reference/keywords/int.md), e deve ter um sem parâmetros, caso contrário, um parâmetro de tipo `string[]`.</span><span class="sxs-lookup"><span data-stu-id="4de4d-106">`Main` must be declared as static and it must return [void](../../csharp/language-reference/keywords/void.md) or [int](../../csharp/language-reference/keywords/int.md), and it must have either no parameters or else one parameter of type `string[]`.</span></span>
