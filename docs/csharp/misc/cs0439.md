# <a name="compiler-error-cs0439"></a>Erro do Compilador CS0439
Uma declaração de alias extern deve preceder todos os outros elementos definidos no namespace  
  
 Esse erro ocorre `extern` quando uma declaração vem `using` depois de outra coisa, como uma declaração, no mesmo namespace. As `extern` declarações devem vir antes de todos os outros elementos do namespace.  
  
## <a name="example"></a>Exemplo  
 O exemplo a seguir gera CS0439:  
  
```csharp  
// CS0439.cs  
using System;  
  
extern alias MyType;   // CS0439  
// To resolve the error, make the extern alias the first line in the file.  
  
public class Test
{  
    public static void Main()
    {  
    }  
}  
```
