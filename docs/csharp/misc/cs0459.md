## <a name="example"></a><span data-ttu-id="d9c1f-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="d9c1f-107">Example</span></span>

 <span data-ttu-id="d9c1f-108">O exemplo a seguir gera CS0459 quando é feita uma tentativa de obter o endereço de uma variável local somente leitura em um loop `foreach` e em um bloco de instrução `fixed`:</span><span class="sxs-lookup"><span data-stu-id="d9c1f-108">The following example generates CS0459 when an attempt is made to take the address of a read-only local variable in a `foreach` loop and in a `fixed` statement block:</span></span>

```csharp
// CS0459.cs
// compile with: /unsafe

class Program
{
    public unsafe void M1()
    {
        int[] ints = new int[] { 1, 2, 3 };
        foreach (int i in ints)
        {
            int *j = &i;  // CS0459
        }

        fixed (int *i = &_i)
        {
            int **j = &i;  // CS0459
        }
    }

    private int _i = 0;
}
```
