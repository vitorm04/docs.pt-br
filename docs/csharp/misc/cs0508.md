## <a name="example"></a><span data-ttu-id="a7962-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="a7962-106">Example</span></span>  
 <span data-ttu-id="a7962-107">O exemplo a seguir gera CS0508.</span><span class="sxs-lookup"><span data-stu-id="a7962-107">The following sample generates CS0508.</span></span>  
  
```csharp  
// CS0508.cs  
// compile with: /target:library  
abstract public class Clx  
{  
   public int i = 0;  
   // Return type is int.  
   abstract public int F();  
}  
  
public class Cly : Clx  
{  
   public override double F()  
   {  
      return 0.0;   // CS0508  
   }  
}  
```
