## <a name="example"></a><span data-ttu-id="df9be-111">Exemplo</span><span class="sxs-lookup"><span data-stu-id="df9be-111">Example</span></span>

```csharp

    {  
        int num;  
        public void TestMethod()  
        {  
            num = 5; // CS0844  
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }
```

## <a name="correct-the-error-by-using-thisnum-to-refer-to-the-class-field"></a><span data-ttu-id="df9be-113">Corrija o erro usando `this.num` para se referir ao campo de classe</span><span class="sxs-lookup"><span data-stu-id="df9be-113">Correct the error by using `this.num` to refer to the class field</span></span>

```csharp

    {  
        int num;  
        public void TestMethod()  
        {  
            this.num = 5; // Error fixed.
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
