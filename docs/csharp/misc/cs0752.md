# <a name="compiler-error-cs0752"></a><span data-ttu-id="37134-102">CS0752 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="37134-102">Compiler Error CS0752</span></span>
<span data-ttu-id="37134-103">Um método parcial não pode ter parâmetros out</span><span class="sxs-lookup"><span data-stu-id="37134-103">A partial method cannot have out parameters</span></span>  
  
 <span data-ttu-id="37134-104">Um método parcial não pode ter um [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parâmetro.</span><span class="sxs-lookup"><span data-stu-id="37134-104">A partial method cannot have an [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="37134-105">Parâmetros de saída não são permitidas porque se o método parcial é removido pelo compilador e não há nenhuma garantia de que o parâmetro de saída é atribuído.</span><span class="sxs-lookup"><span data-stu-id="37134-105">Out parameters are not allowed because if the partial method is removed by the compiler then there is no guarantee that the out parameter is ever assigned.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="37134-106">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="37134-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="37134-107">Remova o modificador de fora do parâmetro e use o valor de retorno do método em vez disso, caso contrário, remova o modificador parcial na declaração de método.</span><span class="sxs-lookup"><span data-stu-id="37134-107">Remove the out modifier from the parameter and use the return value of the method instead, or else remove the partial modifier from the method declaration.</span></span>
