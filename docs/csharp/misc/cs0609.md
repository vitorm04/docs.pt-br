# CS0609 de erro do compilador
Não é possível definir o IndexerName atributo em um indexador marcado como substituir  
  
 O atributo de nome \(**IndexerNameAttribute**\) não pode ser aplicado a uma propriedade indexada é uma substituição. Para obter mais informações, consulte [indexadores](../../csharp/programming-guide/indexers/index.md).  
  
 O exemplo a seguir gera CS0609:  
  
```  
// CS0609.cs using System; using System.Runtime.CompilerServices; public class idx { public virtual int this[int iPropIndex] { get { return 0; } set { } } } public class MonthDays : idx { [IndexerName("MonthInfoIndexer")]   // CS0609, delete to resolve this CS0609 public override int this[int iPropIndex] { get { return 0; } set { } } } public class test { public static void Main( string[] args ) { } }  
```
