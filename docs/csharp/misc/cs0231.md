# <a name="compiler-error-cs0231"></a><span data-ttu-id="1e31d-102">Erro do Compilador CS0231</span><span class="sxs-lookup"><span data-stu-id="1e31d-102">Compiler Error CS0231</span></span>
<span data-ttu-id="1e31d-103">Um parâmetro de parâmetros deve ser o último parâmetro em uma lista formal de parâmetros.</span><span class="sxs-lookup"><span data-stu-id="1e31d-103">A params parameter must be the last parameter in a formal parameter list.</span></span>  
  
 <span data-ttu-id="1e31d-104">O [parâmetro params](../language-reference/keywords/params.md) suporta um número variável de argumentos e deve estar atrás de todos os outros parâmetros.</span><span class="sxs-lookup"><span data-stu-id="1e31d-104">The [params](../language-reference/keywords/params.md) parameter supports a variable numbers of arguments and must be after all other parameters.</span></span> <span data-ttu-id="1e31d-105">Para saber mais, veja [Métodos](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="1e31d-105">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="1e31d-106">A seguinte amostra gera CS0231:</span><span class="sxs-lookup"><span data-stu-id="1e31d-106">The following sample generates CS0231:</span></span>  
  
```csharp  
// CS0231.cs  
class Test  
{  
   public void TestMethod(params int[] p, int i) {} // CS0231  
   // To resolve the error, use the following line instead:  
   // public void TestMethod(int i, params int[] p) {}
  
   static void Main()
   {  
   }  
}  
```
