# <a name="compiler-warning-level-3-cs0219"></a>Compilador aviso (nível 3) CS0219
A variável 'variável' é atribuída mas seu valor nunca é usado  
  
 O compilador emite um aviso de três níveis, quando você declarar e atribuir uma variável, mas não usá-lo.  
  
 O exemplo a seguir gera CS0219:  
  
```csharp  
// CS0219.cs  
// compile with: /W:3  
public class MyClass  
{  
   public static void Main()  
   {  
      int a = 0;   // CS0219  
   }  
}  
```
