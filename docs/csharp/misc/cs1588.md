# <a name="compiler-error-cs1588"></a>CS1588 de erro do compilador
Não é possível determinar o diretório de tempo de execução de linguagem comum – 'motivo'  
  
 O compilador não pode recuperar o diretório que o common language runtime do .NET Framework é instalado em. Esse erro indica que o common language runtime está instalado incorretamente.
