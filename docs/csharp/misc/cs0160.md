# <a name="compiler-error-cs0160"></a><span data-ttu-id="9aa65-103">Erro do Compilador CS0160</span><span class="sxs-lookup"><span data-stu-id="9aa65-103">Compiler Error CS0160</span></span>
<span data-ttu-id="9aa65-104">Uma cláusula catch anterior já captura todas as exceções deste ou de um tipo super (' type ')</span><span class="sxs-lookup"><span data-stu-id="9aa65-104">A previous catch clause already catches all exceptions of this or of a super type ('type')</span></span>  
  
<span data-ttu-id="9aa65-105">Uma série de instruções [Catch](../language-reference/keywords/try-catch.md) precisa estar em ordem decrescente de derivação.</span><span class="sxs-lookup"><span data-stu-id="9aa65-105">A series of [catch](../language-reference/keywords/try-catch.md) statements needs to be in decreasing order of derivation.</span></span> <span data-ttu-id="9aa65-106">Por exemplo, os objetos mais derivados devem aparecer primeiro.</span><span class="sxs-lookup"><span data-stu-id="9aa65-106">For example, the most derived objects must appear first.</span></span>
  
 <span data-ttu-id="9aa65-107">Para obter mais informações, consulte [exceções e tratamento de exceções](../programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="9aa65-107">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="9aa65-108">O exemplo a seguir gera CS0160:</span><span class="sxs-lookup"><span data-stu-id="9aa65-108">The following sample generates CS0160:</span></span>  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
