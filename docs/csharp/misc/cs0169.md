# <a name="compiler-warning-level-3-cs0169"></a>Compilador aviso (nível 3) CS0169
Campo particular 'membro da classe' nunca é usado.  
  
 Uma variável particular foi declarada mas nunca referenciada. Uma maneira comum para gerar esse aviso é quando você declara um membro privado de uma classe e não usá-lo.  
  
 O exemplo a seguir gera CS0169:  
  
```csharp  
// compile with: /W:3  
using System;  
public class ClassX  
{  
   int i;   // CS0169, i is not used anywhere  
  
   public static void Main()  
   {  
   }  
}  
```
