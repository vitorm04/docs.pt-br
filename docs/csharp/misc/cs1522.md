# Compilador CS1522 de aviso (n&#237;vel 1)
Bloco switch vazio  
  
 O compilador detectado um [Alternar](../../csharp/language-reference/keywords/switch.md) bloco sem nenhum **caso** ou **padrão** instrução. Um `switch` bloco deve ter um ou mais **caso** ou **padrão** instruções.  
  
 O exemplo a seguir gera CS1522:  
  
```  
// CS1522.cs // compile with: /W:1 using System; class x { public static void Main() { int i = 6; switch(i)   // CS1522 { // add something to the switch block, for example: /* case (5): Console.WriteLine("5"); return; default: Console.WriteLine("not 5"); return; */ } } }  
```
