# <a name="compiler-error-cs0218"></a>Erro do compilador CS0218
O tipo ('type') deve conter declarações de operador true e false  
  
Se um tipo definido pelo usuário sobrecarregar o [& operador](../language-reference/operators/and-operator.md) ou [ &#124; operador](../language-reference/operators/or-operator.md), ele também deve definir [true e false](../language-reference/keywords/true-false-operators.md) operadores, para fazer o curto-circuito [& & operador](../language-reference/operators/conditional-and-operator.md) ou [ &#124; &#124; operador](../language-reference/operators/conditional-or-operator.md) definidos.
  
 O exemplo a seguir gera CS0218:  
  
```csharp  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
