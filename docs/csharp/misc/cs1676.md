# <a name="compiler-error-cs1676"></a>Erro do compilador CS1676
O parâmetro 'número' deve ser declarado com a palavra-chave 'palavra-chave'  
  
 Esse erro ocorre quando o modificador de tipo de parâmetro em um método anônimo é diferente daquele usado na declaração do delegado que você está convertendo o método.  
  
 O exemplo a seguir gera CS1676:  
  
```csharp  
// CS1676.cs  
delegate void E(ref int i);  
class Errors   
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
