## <a name="example"></a><span data-ttu-id="eea28-109">Exemplo</span><span class="sxs-lookup"><span data-stu-id="eea28-109">Example</span></span>  
 <span data-ttu-id="eea28-110">Este exemplo ilustra o uso de CS0633 a <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span><span class="sxs-lookup"><span data-stu-id="eea28-110">This example illustrates CS0633 using the <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span></span>

```csharp  
// compile with: /target:module  
#define DEBUG  
using System.Runtime.CompilerServices;  
public class Test  
{  
   [IndexerName("Invalid+Identifier")]   // CS0633  
   // try the following line instead  
   // [IndexerName("DEBUG")]  
   public int this[int i]   
   {   
      get { return i; }   
   }  
}  
```
