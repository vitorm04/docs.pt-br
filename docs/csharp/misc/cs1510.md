# CS1510 de erro do compilador
Ref ou out argumento deve ser uma variável pode ser atribuída  
  
 Somente uma variável pode ser passada como um [ref](../../csharp/language-reference/keywords/ref.md) parâmetro em uma chamada de método. Um `ref` valor é como passar um ponteiro.  
  
## Exemplo  
 O exemplo a seguir gera CS1510:  
  
```  
// CS1510.cs public class C { public static int j = 0; public static void M(ref int j) { j++; } public static void Main () { M (ref 2);   // CS1510, can't pass a number as a ref parameter // try the following to resolve the error // M (ref j); } }  
```
