# <a name="compiler-error-cs1918"></a><span data-ttu-id="fc228-102">Erro do compilador CS1918</span><span class="sxs-lookup"><span data-stu-id="fc228-102">Compiler Error CS1918</span></span>
<span data-ttu-id="fc228-103">Membros da propriedade 'name' do tipo 'type' não pode ser atribuídos com um inicializador de objeto porque ele é de um valor de tipo.</span><span class="sxs-lookup"><span data-stu-id="fc228-103">Members of property 'name' of type 'type' cannot be assigned with an object initializer because it is of a value type.</span></span>  
  
 <span data-ttu-id="fc228-104">Esse erro ocorre quando você tenta usar um inicializador de objeto para inicializar as propriedades de um tipo de struct em si for uma propriedade da classe que está sendo inicializada.</span><span class="sxs-lookup"><span data-stu-id="fc228-104">This error occurs when you try to use an object initializer to initialize the properties of a struct type that is itself a property of the class that is being initialized.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="fc228-105">Para corrigir este erro</span><span class="sxs-lookup"><span data-stu-id="fc228-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="fc228-106">Se totalmente, você deve inicializar os campos da propriedade no inicializador de objeto, altere a estrutura para um tipo de classe.</span><span class="sxs-lookup"><span data-stu-id="fc228-106">If you must fully initialize the fields of the property in the object initializer, change the struct to a class type.</span></span> <span data-ttu-id="fc228-107">Caso contrário, inicializam os membros de struct em uma chamada de método separado depois de criar o objeto usando o inicializador de objeto.</span><span class="sxs-lookup"><span data-stu-id="fc228-107">Otherwise, initialize the struct members in a separate method call after you create the object by using the object initializer.</span></span>  
  
## <a name="example"></a><span data-ttu-id="fc228-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="fc228-108">Example</span></span>  
 <span data-ttu-id="fc228-109">O exemplo a seguir gera CS1918:</span><span class="sxs-lookup"><span data-stu-id="fc228-109">The following example generates CS1918:</span></span>  
  
```csharp  
// cs1918.cs  
public struct MyStruct  
{  
    public int i;  
  
}  
public class Test  
{  
    private MyStruct str = new MyStruct();  
    public MyStruct Str  
    {  
        get  
        {  
            return str;  
        }  
    }  
    public static int Main()  
    {  
        Test t = new Test { Str = { i = 1 } }; // CS1918  
        return 0;  
    }  
}  
```
