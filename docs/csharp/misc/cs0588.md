# CS0588 de erro do compilador
Erro interno do compilador: etapa 'LEX'  
  
 Tente determinar se o compilador está falhando devido a sua incapacidade de analisar sintaxe inesperado. Se esse não for o caso, tente [Fale conosco](/visual-studio/ide/talk-to-us).
