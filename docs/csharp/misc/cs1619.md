# <a name="compiler-error-cs1619"></a>CS1619 de erro do compilador
Não é possível criar arquivo temporário 'filename' - motivo  
  
 O compilador não pôde criar um arquivo temporário para o motivo específico (por exemplo, o disco está cheio).
