# <a name="compiler-error-cs0159"></a><span data-ttu-id="db005-102">CS0159 de erro do compilador</span><span class="sxs-lookup"><span data-stu-id="db005-102">Compiler Error CS0159</span></span>
<span data-ttu-id="db005-103">Nenhum rótulo rótulo dentro do escopo da instrução goto</span><span class="sxs-lookup"><span data-stu-id="db005-103">No such label 'label' within the scope of the goto statement</span></span>  
  
 <span data-ttu-id="db005-104">O rótulo referenciado pelo [goto](../../csharp/language-reference/keywords/goto.md) instrução não pôde ser encontrada no escopo do `goto` instrução.</span><span class="sxs-lookup"><span data-stu-id="db005-104">The label referenced by the [goto](../../csharp/language-reference/keywords/goto.md) statement could not be found within the scope of the `goto` statement.</span></span>  
  
 <span data-ttu-id="db005-105">O exemplo a seguir gera CS0159:</span><span class="sxs-lookup"><span data-stu-id="db005-105">The following sample generates CS0159:</span></span>  
  
```csharp  
// CS0159.cs  
public class Class1  
{  
   public static void Main()  
   {  
      int i = 0;  
  
      switch (i)  
      {  
         case 1:  
            goto case 3;   // CS0159, case 3 label does not exist  
         case 2:  
            break;  
      }  
      goto NOWHERE;   // CS0159, NOWHERE label does not exist  
   }  
}  
```
