# <a name="global-c-reference"></a>global (Referência de C#)
Quando a palavra-chave contextual `global` vem antes do [operador ::](../../../csharp/language-reference/operators/namespace-alias-qualifer.md), refere-se ao namespace global, que é o namespace padrão para qualquer programa em C#, que não foi nomeado de outra maneira. Para obter mais informações, consulte [Como usar o alias de namespace global](../../../csharp/programming-guide/namespaces/how-to-use-the-global-namespace-alias.md).
