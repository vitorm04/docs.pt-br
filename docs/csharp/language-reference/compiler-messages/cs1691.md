## <a name="example"></a><span data-ttu-id="7db0e-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="7db0e-106">Example</span></span>

```csharp

    int i = 1;  
    public static void Main()  
    {  
        C myC = new C();

// #pragma warning disable 1645    
        myC.i++;  
#pragma warning restore 151  // CS1691  
// Try the following line instead:  
// #pragma warning restore 1645    
    }  
}  
```
