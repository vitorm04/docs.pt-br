# <a name="compiler-warning-level-1-cs0465"></a><span data-ttu-id="804c3-102">Aviso do compilador (nível 1) CS0465</span><span class="sxs-lookup"><span data-stu-id="804c3-102">Compiler Warning (level 1) CS0465</span></span>
<span data-ttu-id="804c3-103">Apresentar um método 'Finalize' pode interferir na invocação do destruidor.</span><span class="sxs-lookup"><span data-stu-id="804c3-103">Introducing a 'Finalize' method can interfere with destructor invocation.</span></span> <span data-ttu-id="804c3-104">Você pretendia declarar um destruidor?</span><span class="sxs-lookup"><span data-stu-id="804c3-104">Did you intend to declare a destructor?</span></span>

## <a name="example"></a><span data-ttu-id="804c3-107">Exemplo</span><span class="sxs-lookup"><span data-stu-id="804c3-107">Example</span></span>  
 <span data-ttu-id="804c3-108">O exemplo a seguir gera o erro CS0465.</span><span class="sxs-lookup"><span data-stu-id="804c3-108">The following sample generates CS0465.</span></span>  
  
```csharp  
// CS0465.cs  
// compile with: /target:library  
class A  
{  
   public virtual void Finalize() {}   // CS0465  
}  
  
// OK  
class B  
{  
   ~B() {}  
}  
```
