## <a name="example"></a><span data-ttu-id="cc209-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="cc209-108">Example</span></span>

 <span data-ttu-id="cc209-109">O exemplo a seguir gera o erro CS0122:</span><span class="sxs-lookup"><span data-stu-id="cc209-109">The following sample generates CS0122:</span></span>

```csharp
// CS0122.cs
public class MyClass
{
    // Make public to resolve CS0122.
    void MyMethod()
    {
    }
}

public class MyClass2
{
    public static int Main()
    {  
        var a = new MyClass();  
        // MyMethod is private.
        a.MyMethod();   // CS0122
        return 0;
   }
}
```
