# <a name="compiler-warning-level-2-cs0618"></a><span data-ttu-id="e8df8-102">Aviso do compilador (nível 2) CS0618</span><span class="sxs-lookup"><span data-stu-id="e8df8-102">Compiler Warning (level 2) CS0618</span></span>
<span data-ttu-id="e8df8-103">'member' obsoleto: 'text'</span><span class="sxs-lookup"><span data-stu-id="e8df8-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="e8df8-104">Um membro de classe foi marcado com o atributo `Obsolete`, de modo que um aviso será emitido quando o membro da classe for referenciado.</span><span class="sxs-lookup"><span data-stu-id="e8df8-104">A class member was marked with the `Obsolete` attribute, such that a warning will be issued when the class member is referenced.</span></span> <span data-ttu-id="e8df8-105">Para obter mais informações, consulte [Atributos Comuns](../../programming-guide/concepts/attributes/common-attributes.md).</span><span class="sxs-lookup"><span data-stu-id="e8df8-105">For more information, see [Common Attributes](../../programming-guide/concepts/attributes/common-attributes.md).</span></span>  
  
 <span data-ttu-id="e8df8-106">O exemplo a seguir gera o erro CS0618:</span><span class="sxs-lookup"><span data-stu-id="e8df8-106">The following sample generates CS0618:</span></span>  
  
```csharp  
// CS0618.cs  
// compile with: /W:2  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", false)]   // warn if referenced  
   public static void m2()  
   {  
   }  
  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m2();  // CS0618  
   }  
}  
```
