 <span data-ttu-id="57804-104">Um campo em uma classe não está automaticamente disponível para uma classe aninhada.</span><span class="sxs-lookup"><span data-stu-id="57804-104">A field in a class is not automatically available to a nested class.</span></span> <span data-ttu-id="57804-105">Para estar disponível para uma classe aninhada, o campo deve ser [estático](../../../csharp/language-reference/keywords/static.md).</span><span class="sxs-lookup"><span data-stu-id="57804-105">To be available to a nested class, the field must be [static](../../../csharp/language-reference/keywords/static.md).</span></span> <span data-ttu-id="57804-106">Caso contrário, será necessário criar uma instância da classe externa.</span><span class="sxs-lookup"><span data-stu-id="57804-106">Otherwise, you must create an instance of the outer class.</span></span> <span data-ttu-id="57804-107">Para obter mais informações, consulte [Tipos aninhados](../../../csharp/programming-guide/classes-and-structs/nested-types.md).</span><span class="sxs-lookup"><span data-stu-id="57804-107">For more information, see [Nested Types](../../../csharp/programming-guide/classes-and-structs/nested-types.md).</span></span>

```csharp  
   public int count;  
   // try the following line instead  
   // public static int count;  
  
   class InnerClass  
   {  
      void func()  
      {  
         // or, create an instance  
         // OuterClass class_inst = new OuterClass();  
         // int count2 = class_inst.count;  
         int count2 = count;   // CS0038  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
