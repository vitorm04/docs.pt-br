## <a name="example"></a><span data-ttu-id="ab775-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="ab775-108">Example</span></span>  
 <span data-ttu-id="ab775-109">Este exemplo de código consiste em um arquivo .cpp, que é compilado em um arquivo .dll e um arquivo .cs, que usa esse .dll.</span><span class="sxs-lookup"><span data-stu-id="ab775-109">This code sample consists of a .cpp file, which compiles to a .dll, and a .cs file, which uses that .dll.</span></span> <span data-ttu-id="ab775-110">O código a seguir é para o arquivo .dll e define uma propriedade a ser acessada pelo código no arquivo .cs.</span><span class="sxs-lookup"><span data-stu-id="ab775-110">The following code is for the .dll file and defines a property to be accessed by the code in the .cs file.</span></span>  
  
```cpp  
// CPP1546.cpp  
// compile with: /clr /LD  
using namespace System;  
public ref class MCPP  
{  
public:  
    property int Prop [int,int]  
    {  
        int get( int i, int b )  
        {  
            return i;  
        }  
    }  
};  
```
