# <a name="compiler-error-cs1674"></a><span data-ttu-id="7a972-102">Erro do Compilador CS1674</span><span class="sxs-lookup"><span data-stu-id="7a972-102">Compiler Error CS1674</span></span>
<span data-ttu-id="7a972-103">'T': tipo usado em uma instrução using deve ser implicitamente conversível para 'System.IDisposable'</span><span class="sxs-lookup"><span data-stu-id="7a972-103">'T': type used in a using statement must be implicitly convertible to 'System.IDisposable'</span></span>  
  
 <span data-ttu-id="7a972-104">A [instrução using](../../../csharp/language-reference/keywords/using-statement.md) destina-se a ser usada para garantir o descarte de um objeto ao final do bloco `using`, portanto, somente os tipos que são descartáveis podem ser usados nesse tipo de instrução.</span><span class="sxs-lookup"><span data-stu-id="7a972-104">The [using Statement](../../../csharp/language-reference/keywords/using-statement.md) is intended to be used to ensure the disposal of an object at the end of the `using` block, thus, only types which are disposable may be used in such a statement.</span></span> <span data-ttu-id="7a972-105">Por exemplo, tipos de valor não são descartáveis e parâmetros de tipo que não são restritos a serem classes podem não ser considerados descartáveis.</span><span class="sxs-lookup"><span data-stu-id="7a972-105">For example, value types are not disposable, and type parameters which are not constrained to be classes may not be assumed to be disposable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7a972-106">Exemplo</span><span class="sxs-lookup"><span data-stu-id="7a972-106">Example</span></span>

      a++;

## <a name="example"></a><span data-ttu-id="7a972-108">Exemplo</span><span class="sxs-lookup"><span data-stu-id="7a972-108">Example</span></span>  
 <span data-ttu-id="7a972-109">O exemplo a seguir gera o erro CS1674.</span><span class="sxs-lookup"><span data-stu-id="7a972-109">The following sample generates CS1674.</span></span>  
  
```csharp  
// CS1674_b.cs  
using System;  
class C {  
   public void Test() {  
      using (C c = new C()) {}   // CS1674  
   }

class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}

      using (D d = new D()) {}  
   }

```
