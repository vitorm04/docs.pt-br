# <a name="compiler-error-cs0552"></a><span data-ttu-id="86106-102">Erro do Compilador CS0552</span><span class="sxs-lookup"><span data-stu-id="86106-102">Compiler Error CS0552</span></span>
<span data-ttu-id="86106-103">'rotina de conversão': conversão definida pelo usuário de/para interface</span><span class="sxs-lookup"><span data-stu-id="86106-103">'conversion routine' : user defined conversion to/from interface</span></span>  
  
 <span data-ttu-id="86106-104">Não é possível criar conversões definidas pelo usuário para ou de uma interface.</span><span class="sxs-lookup"><span data-stu-id="86106-104">You cannot create a user-defined conversion to or from an interface.</span></span> <span data-ttu-id="86106-105">Caso a rotina de conversão seja necessária, resolva esse erro transformando a interface em uma classe ou derivando uma classe da interface.</span><span class="sxs-lookup"><span data-stu-id="86106-105">If you need the conversion routine, resolve this error by making the interface a class or derive a class from the interface.</span></span>  
  
 <span data-ttu-id="86106-106">O exemplo a seguir gera o erro CS0552:</span><span class="sxs-lookup"><span data-stu-id="86106-106">The following sample generates CS0552:</span></span>  
  
```csharp  
// CS0552.cs  
public interface ii  
{  
}  
  
public class a  
{  
   // delete the routine to resolve CS0552  
   public static implicit operator ii(a aa) // CS0552  
   {  
      return new ii();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
