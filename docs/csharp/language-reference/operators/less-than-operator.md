# <a name="lt-operator-c-reference"></a>Operador &lt; (Referência de C#)
Todos os tipos numéricos e de enumeração definem um operador relacional "menor que" (`<`) que retornará `true` se o primeiro operando for menor que o segundo, `false` caso contrário.  
  
## <a name="remarks"></a>Comentários  
 Os tipos definidos pelo usuário podem sobrecarregar o operador `<` (consulte [operador](../../../csharp/language-reference/keywords/operator.md)). Se `<` estiver sobrecarregado, [>](../../../csharp/language-reference/operators/greater-than-operator.md) também deverá estar sobrecarregado.
