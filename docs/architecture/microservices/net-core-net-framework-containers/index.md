# <a name="choosing-between-net-core-and-net-framework-for-docker-containers"></a>Escolhendo entre o .NET Core e do .NET Framework para contêineres do Docker

Há suporte para duas estruturas para criar aplicativos do Docker em contêineres no lado do servidor com o .NET: [.NET Framework e .NET Core](https://dotnet.microsoft.com/download). Eles compartilham muitos componentes da plataforma .NET e você pode compartilhar o código entre os dois. No entanto, há diferenças fundamentais entre eles e a estrutura a ser usada dependerá do que você deseja realizar. Esta seção fornece orientações sobre quando escolher cada estrutura.
