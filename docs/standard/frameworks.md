<span data-ttu-id="5bdfb-112">Para estruturas de destino baseadas em pacote (por exemplo, .NET 5, .NET Core e .NET Standard), as APIs são definidas pelos pacotes incluídos no aplicativo ou na biblioteca.</span><span class="sxs-lookup"><span data-stu-id="5bdfb-112">For package-based target frameworks (for example, .NET 5, .NET Core, and .NET Standard), the APIs are defined by the packages included in the app or library.</span></span> <span data-ttu-id="5bdfb-113">Um *metapacote* é um pacote NuGet que não tem nenhum conteúdo próprio, mas é uma lista de dependências (outros pacotes).</span><span class="sxs-lookup"><span data-stu-id="5bdfb-113">A *metapackage* is a NuGet package that has no content of its own but is a list of dependencies (other packages).</span></span> <span data-ttu-id="5bdfb-114">Uma estrutura de destino com base em pacote NuGet especifica implicitamente um metapacote que faz referência a todos os pacotes que, juntos, compõem a estrutura.</span><span class="sxs-lookup"><span data-stu-id="5bdfb-114">A NuGet package-based target framework implicitly specifies a metapackage that references all the packages that together make up the framework.</span></span>
