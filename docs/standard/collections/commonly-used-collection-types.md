 <span data-ttu-id="7b83e-104">Coleções são baseadas nas interfaces <xref:System.Collections.ICollection>, <xref:System.Collections.IList>, <xref:System.Collections.IDictionary> ou em seus  <span data-ttu-id="7b83e-109">Coleções genéricas são a melhor solução para tipagem forte.</span><span class="sxs-lookup"><span data-stu-id="7b83e-109">Generic collections are the best solution to strong typing.</span></span> <span data-ttu-id="7b83e-110">No entanto, se o idioma não oferecer suporte a genéricos, o namespace <xref:System.Collections> incluirá coleções base, tais como <xref:System.Collections.CollectionBase>, <xref:System.Collections.ReadOnlyCollectionBase> e <xref:System.Collections.DictionaryBase>, que são classes base abstratas que podem ser estendidas para criar classes de coleção que são fortemente tipadas.</span><span class="sxs-lookup"><span data-stu-id="7b83e-110">However, if your language does not support generics, the <xref:System.Collections> namespace includes base collections, such as <xref:System.Collections.CollectionBase>, <xref:System.Collections.ReadOnlyCollectionBase>, and <xref:System.Collections.DictionaryBase>, which are abstract base classes that can be extended to create collection classes that are strongly typed.</span></span> <span data-ttu-id="7b83e-111">Quando for necessário acesso de coleção com multithread eficiente, use as coleções genéricas no namespace <xref:System.Collections.Concurrent>.</span><span class="sxs-lookup"><span data-stu-id="7b83e-111">When efficient multi-threaded collection access is required, use the generic collections in the <xref:System.Collections.Concurrent> namespace.</span></span>

class.</span></span> <span data-ttu-id="7b83e-115">Os elementos de uma <xref:System.Collections.Hashtable> ou uma <xref:System.Collections.Generic.Dictionary%602> são acessíveis somente pela chave do elemento.</span><span class="sxs-lookup"><span data-stu-id="7b83e-115">The elements of a <xref:System.Collections.Hashtable> or a <xref:System.Collections.Generic.Dictionary%602> are accessible only the element.</span></span> <span data-ttu-id="7b83e-116">Os índices em todas as coleções são baseados em zero, exceto <xref:System.Array>, que permite matrizes que não sejam baseadas em zero.</span><span class="sxs-lookup"><span data-stu-id="7b83e-116">The indexes in all collections are zero-based, except <xref:System.Array>, which allows arrays that are not zero-based.</span></span>  <span data-ttu-id="7b83e-117">O recurso LINQ para objetos permite que você use consultas LINQ para acessar objetos na memória, desde que o tipo de objeto implemente <xref:System.Collections.IEnumerable> ou <xref:System.Collections.Generic.IEnumerable%601>.</span><span class="sxs-lookup"><span data-stu-id="7b83e-117">The LINQ to Objects feature allows you to use LINQ queries to access in-memory objects as long as the object type implements <xref:System.Collections.IEnumerable> or <xref:System.Collections.Generic.IEnumerable%601>.</span></span> <span data-ttu-id="7b83e-118">Consultas LINQ fornecem um padrão comum para acessar dados; são geralmente mais concisas e legíveis que os loops `foreach` padrão; e fornecem recursos de filtragem, classificação e agrupamento.</span><span class="sxs-lookup"><span data-stu-id="7b83e-118">LINQ queries provide a common pattern for accessing data; are typically more concise and readable than standard `foreach` loops; and provide filtering, ordering and grouping capabilities.</span></span> <span data-ttu-id="7b83e-119">Consultas LINQ também podem melhorar o desempenho.</span><span class="sxs-lookup"><span data-stu-id="7b83e-119">LINQ queries can also improve performance.</span></span> <span data-ttu-id="7b83e-120">Para obter mais informações, consulte [LINQ to Objects](https://msdn.microsoft.com/library/73cafe73-37cf-46e7-bfa7-97c7eea7ced9) e [PLINQ (Parallel LINQ)](../../../docs/standard/parallel-programming/parallel-linq-plinq.md).</span><span class="sxs-lookup"><span data-stu-id="7b83e-120">For more information, see [LINQ to Objects](https://msdn.microsoft.com/library/73cafe73-37cf-46e7-bfa7-97c7eea7ced9) and [Parallel LINQ (PLINQ)](../../../docs/standard/parallel-programming/parallel-linq-plinq.md).</span></span>
