```vb
Try
    [ tryStatements ]
    [ Exit Try ]
[ Catch [ exception [ As type ] ] [ When expression ]
    [ catchStatements ]

[ Catch ... ]
[ Finally
    [ finallyStatements ] ]
End Try
```
