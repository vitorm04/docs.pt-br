# <a name="mycomputernetwork-object"></a>Objeto My.Computer.Network
Fornece uma propriedade, um evento e métodos para interagir com a rede à qual o computador está conectado.

 Para obter informações sobre os métodos e propriedades do `My.Computer.Network` de objeto, consulte <xref:Microsoft.VisualBasic.Devices.Network>.</xref:Microsoft.VisualBasic.Devices.Network>
