# <a name="type-conversions-in-visual-basic"></a>Conversões de tipo no Visual Basic
O processo de alterar um valor de um tipo de dados para outro tipo é chamado *conversão*. Conversões são *widening* ou *narrowing*, dependendo das capacidades de dados dos tipos envolvidos. Eles também são *implícita* ou *explícita*, dependendo da sintaxe no código-fonte.
