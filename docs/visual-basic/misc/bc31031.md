# <a name="project-level-conditional-compilation-constant-is-not-valid-lterrorgt"></a>Constante de compilação condicional no nível de projeto não é válida: &lt;erro&gt;
Uma opção de compilação condicional inválida for especificada para o projeto.  
  
 **ID do erro:** BC31031  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Verifique se as constantes de compilação condicional usadas com este projeto.
