# <a name="expected-3939-for-xml-end-tag"></a>' /' Esperado em uma marca de fim XML
A marca de fim XML para um literal XML está faltando o obrigatório '/' caracteres.  
  
 **ID do erro:** BC31167  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione o caractere '/' para o início da marca de fim XML para a literal XML.
