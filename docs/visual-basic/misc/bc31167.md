# <a name="expected-3939-for-xml-end-tag"></a>Esperado &#39; / &#39; para a marca de fim XML
A marca de fim XML para um literal XML está faltando o necessária '/' caracteres.  
  
 **ID do erro:** BC31167  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione o caractere '/' para o início da marca de fim XML para o XML literal.
