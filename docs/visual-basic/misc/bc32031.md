# &#39;End Sub&#39; deve ser a primeira instru&#231;&#227;o em uma linha
Um `End Sub` instrução segue um separador de declaração dois\-pontos \(:\).`End Sub` deve ser a única declaração na sua linha de código.  
  
 **ID do erro:** BC32031  
  
### Para corrigir este erro  
  
-   Separe as várias declarações em linhas diferentes.
