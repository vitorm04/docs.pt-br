# <a name="39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-not-a-class"></a>'&lt;typename&gt;' não pode ser usado como um atributo porque ele não é uma classe
Foi feita uma tentativa para usar um tipo que não é uma classe como um atributo.  
  
 **ID do erro:** BC31503  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Defina atributos personalizados como classes que derivam de `System.Attribute`.
