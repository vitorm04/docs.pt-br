# &#39;Is&#39; esperado
Um `TypeOf` operador ocorre sem um `Is` cláusula.  
  
 **ID do erro:** BC30224  
  
### Para corrigir este erro  
  
-   Adicione um `Is` seguinte cláusula de `TypeOf` operador; por exemplo, `TypeOf MyControl Is ComboBox`.
