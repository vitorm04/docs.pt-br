# <a name="is-expected"></a>'Is' esperado
Um `TypeOf` operador ocorre sem um `Is` cláusula.  
  
 **ID do erro:** BC30224  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar um `Is` seguinte cláusula de `TypeOf` operador; por exemplo, `TypeOf MyControl Is ComboBox`.
