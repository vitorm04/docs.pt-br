# <a name="39namespace39-statement-must-end-with-a-matching-39end-namespace39"></a>A instrução 'Namespace' deve finalizar com 'End Namespace' correspondente
`Namespace`blocos devem terminar com um `End Namespace` instrução.  
  
 **ID do erro:** BC30626  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar uma `End Namespace` instrução ao final do `Namespace` bloco.
