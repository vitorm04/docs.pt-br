# Instru&#231;&#245;es &#39;Error&#39; n&#227;o s&#227;o v&#225;lidas na janela imediata
`Error` instruções só podem ser usadas no código\-fonte.  
  
 **ID do erro:** BC30724  
  
### Para corrigir este erro  
  
-   Remover `Error` instruções do código de depuração.
