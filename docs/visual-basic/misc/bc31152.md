# <a name="xml-namespace-prefix-prefix-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>Prefixo de namespace XML 'prefixo' está reservado para uso por XML e o namespace que URI não pode ser alterado.
Um prefixo de namespace XML, de reservados como `xmlns`, foi usado em um literal XML.  
  
 **ID do erro:** BC31152  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere o prefixo de namespace XML para um valor que não está reservado.
