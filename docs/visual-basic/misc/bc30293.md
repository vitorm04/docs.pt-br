# <a name="lterrorgt-39ltstructurename1gt39-contains-39ltstructurename2gt39"></a>&lt;Erro&gt;: &#39;&lt; structurename1&gt;&#39; contém &#39;&lt; structurename2&gt;&#39;
Uma definição de estrutura circular ocorre quando uma estrutura designa a mesmo como um de seus próprios membros.  
  
 **ID do erro:** BC30293  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere o nome do membro da estrutura.
