# N&#227;o &#233; poss&#237;vel ler campos delimitados porque Delimiters &#233; Nothing ou vazio
O `TextFieldParser` não pode ler o arquivo porque o `Delimiters` está definida como `Nothing` ou está vazia.  
  
### Para corrigir este erro  
  
-   Forneça um valor válido para `Delimiters`.
