# <a name="expected-matching-closing-single-quote-for-xml-attribute-value"></a>Aspa simples de valor do atributo XML de fechamento esperadas
Um valor de atributo que é delimitado por aspas simples não tem o fechamento caractere de aspas.  
  
 **ID do erro:** BC31163  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione o caractere de aspas de fechamento (') para delimitar o valor do atributo.
