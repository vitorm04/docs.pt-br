# <a name="39lttypenamegt39-must-be-declared-39mustinherit39-because-it-contains-methods-declared-39mustoverride39"></a>'&lt;typename&gt;' deve ser declarado 'MustInherit' porque ele contém métodos declarados 'MustOverride'
Tipos que contenham `MustOverride` membros devem ser declarados como `MustInherit`.  
  
 **ID do erro:** BC31411  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar o `MustInherit` modificador no tipo.
