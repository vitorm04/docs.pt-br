# &#39;AddHandler&#39; j&#225; foi declarado.
Mais de um `AddHandler` declaração aparece em uma declaração de evento personalizado. Um `AddHandler` declaração declara procedimento usado para adicionar um manipulador de eventos.  
  
 **ID do erro:** BC31127  
  
### Para corrigir este erro  
  
-   Remova a `AddHandler` instrução.
