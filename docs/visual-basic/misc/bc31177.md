# Espa&#231;o em branco n&#227;o pode aparecer aqui
Um ou mais espaços são incluídos no nome de um literal de elemento XML.  
  
 **ID do erro:** BC31177  
  
### Para corrigir este erro  
  
-   Remova o nome do elemento XML literal espaço ou espaços.
