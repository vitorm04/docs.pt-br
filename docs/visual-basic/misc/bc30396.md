# <a name="specifier-is-not-valid-on-an-enum-declaration"></a>'\<especificador >' não é válido em uma declaração de enumeração
Você usou um especificador que não é válido em um `Enum` declaração.  
  
 **ID do erro:** BC30396  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remova o especificador.
