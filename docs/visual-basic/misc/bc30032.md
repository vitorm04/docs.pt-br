# Eventos n&#227;o podem ter um tipo de retorno
Eventos podem aceitar argumentos, mas não podem retornar valores diretamente.  
  
 **ID do erro:** BC30032  
  
### Para corrigir este erro  
  
-   Remova o tipo de retorno do `Event` instrução.
