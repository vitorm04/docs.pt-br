# <a name="initialization-error"></a>Erro de inicialização
Ocorreu um erro ao inicializar o programa. Isso pode ser um resultado de cancelar uma instalação enquanto ele ainda está em andamento.  
  
 **ID do erro:** BC2000  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Tente a instalação novamente.
