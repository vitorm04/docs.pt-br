# Tipo de pelo menos um par&#226;metro deste operador bin&#225;rio deve ser do tipo recipiente &#39;&lt; typename &gt;&#39;
Uma definição de um operador binário especifica ambos os parâmetros com tipos diferentes de classe ou estrutura na qual o operador está definido.  
  
 Quando você define um operador em uma classe ou estrutura, pelo menos um dos parâmetros deve ser do tipo de classe ou estrutura.  
  
 **ID do erro:** BC33021  
  
### Para corrigir este erro  
  
-   Altere o tipo de um ou ambos os parâmetros para o tipo de classe ou estrutura na qual o operador está definido.
