# <a name="vbstrconvwide-and-vbstrconvnarrow-cannot-be-combined"></a>VbStrConv. Wide e VbStrConv. Narrow não podem ser combinados
Seu aplicativo está tentando combinar os `VbStrConv` membros de enumeração `Wide` e `Narrow`, que são mutuamente exclusivo.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remova `VbStrConv.Wide` ou `VbStrConv.Narrow`.
