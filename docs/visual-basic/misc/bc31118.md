# <a name="end-addhandler-must-be-the-first-statement-on-a-line"></a>'End AddHandler' deve ser a primeira instrução em uma linha
Um `End AddHandler` instrução segue um separador de instrução de dois-pontos (:). `End AddHandler` deve ser a única instrução em sua linha de código-fonte.  
  
 **ID do erro:** BC31118  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Separe várias declarações em linhas diferentes.
