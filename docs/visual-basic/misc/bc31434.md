# <a name="method-39ltmethodname1gt39-cannot-implement-partial-method-39ltmethodname2gt39-because-39ltmethodname3gt39-already-implements-it"></a>Método &#39; &lt;methodname1&gt;&#39; não pode implementar o método parcial &#39;&lt; methodname2&gt;&#39; porque &#39;&lt; methodname3&gt;&#39; já o implementa
Método '\<methodname1 >' não pode implementar o método parcial '\<methodname2 >' porque '\<methodname3 >' já o implementa. Somente um método pode implementar um método parcial.  
  
 Você não pode ter dois métodos parciais que implementam a mesma declaração de método parcial. O código a seguir faz com que esse erro.  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub ValueChanged()  
    End Sub  
  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' First implementation of the partial method.  
    Private Sub ValueChanged()  
        MsgBox(Value was changed to " & Me.Quantity)  
    End Sub  
  
    ' Second implementation of the partial method causes this error.  
    'Private Sub ValueChanged()  
    '    Console.WriteLine("Quantity was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```  
  
 **ID do erro:** BC31434
