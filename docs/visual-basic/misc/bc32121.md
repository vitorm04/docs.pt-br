# <a name="cannot-inherit-interface-interfacename1-because-the-interface-interfacename2-from-which-it-inherits-could-be-identical-to-interface-interfacename3-for-some-type-arguments"></a><span data-ttu-id="769e3-102">Não é possível herdar a interface ' \<interfacename1 > ' porque a interface ' \<interfacename2 > ' da qual ela herda pode ser idêntica à interface ' \<interfacename3 > ' para alguns argumentos de tipo</span><span class="sxs-lookup"><span data-stu-id="769e3-102">Cannot inherit interface '\<interfacename1>' because the interface '\<interfacename2>' from which it inherits could be identical to interface '\<interfacename3>' for some type arguments</span></span>
<span data-ttu-id="769e3-103">Uma interface genérica herda de duas ou mais interfaces genéricas, e duas das heranças podem entrar em conflito para determinados valores dos argumentos de tipo.</span><span class="sxs-lookup"><span data-stu-id="769e3-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="769e3-104">As instruções a seguir podem gerar esse erro.</span><span class="sxs-lookup"><span data-stu-id="769e3-104">The following statements can generate this error.</span></span>  
  
```vb  
Public Interface interfaceA(Of u)  
    Inherits interfaceX(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="769e3-105">Se `derivedInterface` for construído ou implementado fornecendo o mesmo tipo para `t1` e `t2`, ele deverá herdar duas versões de `interfaceX` com argumentos de tipo idênticos.</span><span class="sxs-lookup"><span data-stu-id="769e3-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceX` with identical type arguments.</span></span> <span data-ttu-id="769e3-106">Isso produziria uma ambiguidade sobre qual versão acessar.</span><span class="sxs-lookup"><span data-stu-id="769e3-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="769e3-107">**ID do erro:** BC32121</span><span class="sxs-lookup"><span data-stu-id="769e3-107">**Error ID:** BC32121</span></span>
