# <a name="end-synclock-must-be-preceded-by-a-matching-synclock"></a>'End SyncLock' deve ser precedido por um 'SyncLock' correspondente
`SyncLock` blocos começam com o `SyncLock` palavra-chave e terminam com o `End SyncLock` construir.  
  
 **ID do erro:** BC30674  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Certifique-se de que o `SyncLock` bloco começa com um `SyncLock` instrução.
