# <a name="end-set-must-be-the-first-statement-on-a-line"></a>'End Set' deve ser a primeira instrução em uma linha
Um `End Set` instrução segue um separador de instrução de dois-pontos (:). `End Set` deve ser a única instrução em sua linha de código-fonte.  
  
 **ID do erro:** BC32034  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Separe várias declarações em linhas diferentes.
