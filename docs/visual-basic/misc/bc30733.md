# <a name="39ltmembernamegt39-is-already-declared-by-39ltprocedurenamegt39-which-was-generated-for-this-ltcontainernamegt"></a>'&lt;membername&gt;'já foi declarado por'&lt;procedurename&gt;', que foi gerado para este &lt;containername&gt;
Um membro foi declarado que tem o mesmo nome como um membro é declarado pelo comom language runtime.  
  
 **ID do erro:** BC30733  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere o nome do membro para remover o conflito.
