# <a name="ltmembernamegt-is-already-declared-by-ltprocedurenamegt-which-was-generated-for-this-ltcontainernamegt"></a>'&lt;membername&gt;'já foi declarado por'&lt;procedurename&gt;', que foi gerado para este &lt;containername&gt;
Um membro foi declarado que tem o mesmo nome que um membro que é declarado pelo tempo de execução de linguagem comom.  
  
 **ID do erro:** BC30733  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Altere o nome do membro para remover o conflito.
