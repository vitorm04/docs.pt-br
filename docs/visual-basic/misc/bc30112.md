# &#39;&lt; typename &gt;&#39; &#233; um namespace e n&#227;o pode ser usado como uma express&#227;o
Um nome de tipo ocorre onde uma expressão é necessária. Uma expressão deve consistir de variáveis, constantes, literais, propriedades, e `Function` chamadas de procedimento.  
  
 **ID do erro:** BC30112  
  
### Para corrigir este erro  
  
-   Remova o nome do tipo e construa a expressão usando elementos válidos.
