# &#39;Next&#39; esperado
Um `On Error Resume` construção ocorre sem um Concluindo `Next`.  
  
 **ID do erro:** BC30003  
  
### Para corrigir este erro  
  
-   Adicionar o `Next` palavra\-chave a fim do `On Error` instrução.
