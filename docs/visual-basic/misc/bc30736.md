# Express&#227;o n&#227;o pode ser avaliada no momento
A expressão especificada não pôde ser avaliada no contexto atual.  
  
 **ID do erro:** BC30736  
  
### Para corrigir este erro  
  
-   Altere a sintaxe da expressão.
