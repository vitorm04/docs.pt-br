# <a name="shared-39sub-new39-cannot-have-any-parameters"></a>Compartilhado 'Sub New' não pode ter nenhum parâmetro
Você incluiu um ou mais parâmetros em uma chave secreta `Sub New` instrução, que não é válida.  
  
 **ID do erro:** BC30479  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o parâmetro ou parâmetros.
