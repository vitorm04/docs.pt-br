# Construtor deve ser declarado como Sub, n&#227;o como uma fun&#231;&#227;o
Você tentou declarar uma `Function New`. Construtor deve ser declarado como `Sub New`.  
  
 **ID do erro:** BC30493  
  
### Para corrigir este erro  
  
-   Use `Sub` em vez de `Function`.
