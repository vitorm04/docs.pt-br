# <a name="name-cannot-expose-type-type1-in-specifier-type2-typename-through-name"></a>'\<Name > ' não pode expor \<o tipo type1 \<> no \<especificador >\<type2 > ' TypeName \<> ' por meio de Name >
Você tentou expor um tipo privado fora da classe `MyClass`pública.  
  
 **ID do erro:** BC30508  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Declare o tipo `Public`.
