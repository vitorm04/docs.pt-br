# <a name="39ltelementnamegt39-is-not-a-parameter-of-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>&#39;&lt;ElementName&gt; &#39; não é um parâmetro de método de extensão &#39; &lt;methodname&gt; &#39; definido em &#39; &lt;typename&gt;&#39;
Uma chamada de procedimento fornece um argumento adicional que não é definido pelo método de extensão.  
  
 **ID do erro:** BC36585  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o argumento supérfluo da lista de argumentos.
