# <a name="property-parameters-cannot-be-declared-byref"></a>\<Propriedade > parâmetros não podem ser declarados como 'ByRef'
Parâmetros para propriedades devem ser declarados usando a `ByVal` modificador.  
  
 **ID do erro:** BC30651  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Definir parâmetros de propriedade como `ByVal`.
