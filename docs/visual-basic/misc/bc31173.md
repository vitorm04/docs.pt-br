# <a name="missing-required-white-space"></a>Espaço em branco obrigatório ausente
Um literal XML foi especificado sem um espaço separador necessário. Um exemplo de um espaço separador necessário é o espaço entre os atributos subsequentes em um elemento XML literal.  
  
 **ID do erro:** BC31173  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicione o separador de espaço necessário.
