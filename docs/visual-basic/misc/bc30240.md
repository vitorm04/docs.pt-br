# <a name="exit-must-be-followed-by-sub-function-property-do-for-while-select-or-try"></a>'Exit' deve ser seguido por 'Sub', 'Function', 'Property', 'Do', 'For', 'While', 'Select' ou 'Try'
Um `Exit` instrução contém uma palavra-chave inválida. `Exit` deve especificar o bloco do qual o controle deve ser transferido para a instrução que segue o bloco; Por exemplo, `End While`.  
  
 **ID do erro:** BC30240  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Adicione a seguinte palavra-chave válida `Exit`, ou remova o `Exit` instrução.
