# <a name="extension-method-methodname-has-type-constraints-that-can-never-be-satisfied"></a><span data-ttu-id="cb1bb-102">O método de extensão ' \<methodname > ' tem restrições de tipo que nunca podem ser satisfeitas</span><span class="sxs-lookup"><span data-stu-id="cb1bb-102">Extension method '\<methodname>' has type constraints that can never be satisfied</span></span>
<span data-ttu-id="cb1bb-103">Os parâmetros de tipo desse método interagem de uma maneira que impede que eles nunca sejam satisfeitos.</span><span class="sxs-lookup"><span data-stu-id="cb1bb-103">The type parameters of this method interact in a way that prevents them from ever being satisfied.</span></span> <span data-ttu-id="cb1bb-104">O método de extensão a seguir é um exemplo.</span><span class="sxs-lookup"><span data-stu-id="cb1bb-104">The following extension method is an example.</span></span>  
  
```vb  
'' Not valid.  
'<Extension()> _  
'Sub extensionExample(Of T As U, U)(ByVal para1 As T, ByVal para2 As U)  
'End Sub  
```  
  
 <span data-ttu-id="cb1bb-105">Como o método é um método de extensão, o compilador deve ser capaz de determinar o tipo de dados ou tipos que o método estende com base apenas no primeiro parâmetro na declaração do método, `para1` e o argumento enviado para esse parâmetro.</span><span class="sxs-lookup"><span data-stu-id="cb1bb-105">Because the method is an extension method, the compiler must be able to determine the data type or types that the method extends based only on the first parameter in the method declaration, `para1`, and the argument sent in for that parameter.</span></span> <span data-ttu-id="cb1bb-106">Quando o primeiro parâmetro se refere a parâmetros de tipo genérico, `para1 as T`, as restrições nos parâmetros genéricos restringem o conjunto de tipos aos quais o método se aplica.</span><span class="sxs-lookup"><span data-stu-id="cb1bb-106">When the first parameter refers to generic type parameters, `para1 as T`, the constraints on the generic parameters restrict the set of types to which the method applies.</span></span>  
  
 <span data-ttu-id="cb1bb-107">A aplicabilidade de um método de extensão é determinada do argumento fornecido para o primeiro parâmetro, que é `arg1` no código a seguir.</span><span class="sxs-lookup"><span data-stu-id="cb1bb-107">Applicability of an extension method is determined from the argument provided for the first parameter, which is `arg1` in the following code.</span></span>  
  
 `'' Not valid.`  
  
 `'arg1.extensionExample(arg2)`  
  
 <span data-ttu-id="cb1bb-108">Deve ser possível verificar as restrições em todos os parâmetros de tipo genérico referenciados pelo primeiro parâmetro, `para1`, examinando apenas o primeiro argumento, `arg1`.</span><span class="sxs-lookup"><span data-stu-id="cb1bb-108">It must be possible to verify the constraints on all generic type parameters referred to by the first parameter, `para1`, by looking at only the first argument, `arg1`.</span></span> <span data-ttu-id="cb1bb-109">No `extensionExample`, o conjunto de tipos que está sendo estendido não pode ser determinado apenas a partir do primeiro parâmetro.</span><span class="sxs-lookup"><span data-stu-id="cb1bb-109">In `extensionExample`, the set of types that is being extended cannot be determined from the first parameter alone.</span></span> <span data-ttu-id="cb1bb-110">O parâmetro de tipo `T` é restrito pelo parâmetro de tipo `U`, que não é referenciado por `para1` e não pode ser inferido de `arg1`.</span><span class="sxs-lookup"><span data-stu-id="cb1bb-110">Type parameter `T` is constrained by type parameter `U`, which is not referenced by `para1` and cannot be inferred from `arg1`.</span></span> <span data-ttu-id="cb1bb-111">Portanto, a aplicabilidade do método para qualquer tipo possível não pode ser verificada, e o método nunca pode ser chamado.</span><span class="sxs-lookup"><span data-stu-id="cb1bb-111">Therefore, the applicability of the method to any possible type cannot be verified, and the method can never be called.</span></span>  
  
 <span data-ttu-id="cb1bb-112">**ID do erro:** BC36561</span><span class="sxs-lookup"><span data-stu-id="cb1bb-112">**Error ID:** BC36561</span></span>
