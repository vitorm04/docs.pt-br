# &#39;(&#39; Expected
Uma lista de argumentos em uma chamada de procedimento não começa com um parêntese de abertura \(`(`\) caracteres.  
  
 **ID do erro:** BC30199  
  
### Para corrigir este erro  
  
-   Adicionar um parêntese de abertura caracteres para o início da lista.
