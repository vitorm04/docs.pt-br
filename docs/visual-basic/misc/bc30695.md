# <a name="lttypenamegt-39ltmembernamegt39-must-be-declared-39shadows39-because-another-member-with-this-name-is-declared-39shadows39"></a>&lt;TypeName&gt; '&lt;membername&gt;' deve ser declarado como 'Shadows' porque outro membro com esse nome é declarado como 'Shadows'
Essa propriedade ou procedimento tem o mesmo nome que outra propriedade ou procedimento marcado com o `Shadows` modificador.  
  
 **ID do erro:** BC30695  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Use o `Shadows` modificador.
