# <a name="xml-processing-instruction-name-39name39-is-not-valid"></a>Nome de instrução de processamento XML &#39;nome&#39; não é válido
O nome fornecido para uma instrução de processamento XML é inválido.  
  
 **ID do erro:** BC31174  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Corrija os erros no nome de instrução de processamento XML ou removê-lo.
