# &#39;Lib&#39; esperado
Um `Declare` declaração especifica um nome de biblioteca sem usar o `Lib` palavra\-chave.  
  
 **ID do erro:** BC30218  
  
### Para corrigir este erro  
  
-   Adicionar o `Lib` palavra\-chave antes do nome da biblioteca.
