# <a name="39ltmodifiergt39-is-not-valid-on-an-event-declaration"></a>'&lt;modificador&gt;' não é válido em uma declaração de evento
Um `Event` instrução contém uma palavra-chave inválida, como `ReadOnly`.  
  
 **ID do erro:** BC30243  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remova a palavra-chave inválida do `Event` instrução.
