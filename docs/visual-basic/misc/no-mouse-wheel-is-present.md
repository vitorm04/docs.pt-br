# <a name="no-mouse-wheel-is-present"></a>Nenhuma roda de mouse está presente
O `My.Computer.Mouse.WheelScrollLines` propriedade foi chamada, mas o rato não tem nenhuma roda de rolagem.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique o `My.Computer.Mouse.WheelExists` propriedade para ver se o mouse tem uma roda de rolagem antes de chamar o `My.Computer.Mouse.WheelScrollLines` propriedade.  
  
     -ou-  
  
-   Instale um mouse com uma roda de rolagem no computador.
