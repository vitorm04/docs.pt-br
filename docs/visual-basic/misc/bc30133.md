# <a name="39goto39-statements-are-not-valid-in-the-immediate-window"></a>Instruções 'GoTo' não são válidas na janela imediata
O `GoTo` executa a ramificação de instrução e não é permitida em um contexto de depuração.  
  
 **ID do erro:** BC30133  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Não emita uma `GoTo` instrução o **imediato** janela.
