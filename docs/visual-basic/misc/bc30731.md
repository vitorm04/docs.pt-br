# <a name="39addressof39-expressions-are-not-valid-in-debug-windows"></a>Expressões 'AddressOf' não são válidas em janelas de depuração
`AddressOf`instruções só podem ser usadas no código-fonte.  
  
 **ID do erro:** BC30731  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Remover `AddressOf` instruções do código de depuração.
