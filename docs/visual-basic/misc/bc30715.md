# Instru&#231;&#245;es &#39;Catch&#39; n&#227;o s&#227;o v&#225;lidas na janela imediata
`Catch` instruções só podem ser usadas no código\-fonte.  
  
 **ID do erro:** BC30715  
  
### Para corrigir este erro  
  
-   Remover `Catch` instruções do código de depuração.
