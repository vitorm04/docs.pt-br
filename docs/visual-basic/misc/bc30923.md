# <a name="39ltnamegt39-cannot-expose-type-39lttypegt39-used-in-the-underlying-delegate-type-39ltdelegatetypegt39-of-the-event-it-is-implementing-to-ltspecifier1gt-39lttype1gt39-through-ltspecifier2gt-39lttype2gt39"></a>'&lt;nome&gt;'não pode expor o tipo'&lt;tipo&gt;'usado no tipo delegado subjacente'&lt;delegatetype&gt;' do evento que está implementando a &lt;specifier1&gt; '&lt;type1&gt;' por meio de &lt;specifier2&gt; '&lt;type2&gt;'
O código expõe um tipo privado fora da classe pública.  
  
 **ID do erro:** BC30923  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Declare o tipo `Public` ou use outro tipo.
