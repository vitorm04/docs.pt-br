# <a name="assembly-assemblyname-cannot-be-created-because-its-path-is-longer-than-259-characters"></a>Assembly '\<assemblyname >' não pode ser criado porque seu caminho tem mais de 259 caracteres
O caminho para esse assembly excede o comprimento máximo de 259 caracteres.  
  
 **ID do erro:** BC31518  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
- Crie um caminho mais curto usando nomes mais curtos ou selecionando uma hierarquia de arquivos iniciante.
