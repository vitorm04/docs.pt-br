# Caractere &#39;character&#39; (0x &lt; hexadecimal &gt;) n&#227;o &#233; permitido em XML literal
Um caractere sem suporte foi usado em um literal XML.  
  
 **ID do erro:** BC31176  
  
### Para corrigir este erro  
  
-   Remova o caractere sem suporte.
