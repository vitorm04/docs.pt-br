# <a name="type-39lttypenamegt39-must-be-a-value-type-or-a-type-argument-constrained-to-39structure39-in-order-to-be-used-with-39nullable39-or-nullable-modifier-3939"></a>Tipo &#39; &lt;typeName&gt; &#39; deve ser um tipo de valor ou um argumento de tipo restrito a &#39;estrutura&#39; para ser usado com &#39;Nullable&#39; ou o modificador anulável &#39;?&#39;
Somente tipos de valor, incluindo estruturas, podem ser declarados como nulos.  
  
```vb  
' Valid.  
Dim n? As Integer  
Dim m As Integer?  
  
' Not valid.  
' Dim p? As Object  
' Dim q As Nullable(Of Object)  
```  
  
 **ID do erro:** BC33101  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remova o '?' ou `Nullable`.  
  
-   Use um tipo de dados de valor.
