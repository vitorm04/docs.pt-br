# <a name="in-expected"></a>'In' esperado
Uma `From` `Aggregate` ou cláusula foi especificada sem um `In` operador. Você usa `In` o operador para identificar a coleta para consultar.  
  
 **ID de erro:** BC36607  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  

1. Adicione `In` o operador e `From` os `Aggregate` campos-chave à cláusula ou. A seguir, é mostrado um exemplo:  

```vb  
Dim names = From pers In people
            Select pers.FirstName, pers.LastName  
```
