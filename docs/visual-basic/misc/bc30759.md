# <a name="referenced-39ltmembernamegt39-has-a-value-of-39nothing39"></a>Referenciado '&lt;membername&gt;' tem um valor de 'Nothing'
O item que está sendo referenciado não foi atribuído um valor, ou ele foi atribuído o valor `Nothing`.  
  
 **ID do erro:** BC30759  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se o nome do item que está escrito e configurado corretamente.
