# <a name="try-must-have-at-least-one-39catch39-or-a-39finally39"></a>Try deve ter pelo menos 'Catch' ou 'Finally'
A `Try` bloco deve incluir uma `Finally` bloco ou pelo menos um `Catch` bloquear antes do `End Try` instrução.  
  
 **ID do erro:** BC30030  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar uma `Catch` ou `Finally` bloquear dentro de `Try` bloco.
