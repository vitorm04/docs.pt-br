# Classes n&#227;o podem ser declaradas &#39;&lt; especificador &gt;&#39;
Você tentou declarar uma `Class` ou `Module` com um especificador inválido.  
  
 **ID do erro:** BC30461  
  
### Para corrigir este erro  
  
-   Remova o especificador.
