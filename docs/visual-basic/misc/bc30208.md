# &#39;Compare&#39; esperado
Um `Option` declaração especifica `Text` ou `Binary`, mas não `Compare`.  
  
 **ID do erro:** BC30208  
  
### Para corrigir este erro  
  
-   Adicionar o `Compare` palavra\-chave para o `Option` instrução; por exemplo, `Option Compare Binary`.
