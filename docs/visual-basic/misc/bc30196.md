# V&#237;rgula esperada
Uma lista de argumentos está faltando uma vírgula \(,\) entre dois dos argumentos.  
  
 **ID do erro:** BC30196  
  
### Para corrigir este erro  
  
-   Verifique se que todos os argumentos são separados por vírgulas.
