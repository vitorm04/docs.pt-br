# <a name="property-39ltpropertynamegt39-with-no-parameters-cannot-be-found"></a>Propriedade '&lt;propertyname&gt;' sem parâmetros não pode ser encontrado
O atributo que está sendo usado tem uma propriedade que requer um ou mais parâmetros.  
  
 **ID do erro:** BC30658  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
1.  Especifica parâmetros para a propriedade.
