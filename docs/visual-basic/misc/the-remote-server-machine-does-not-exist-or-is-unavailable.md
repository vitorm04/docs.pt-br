# <a name="the-remote-server-machine-does-not-exist-or-is-unavailable-visual-basic"></a>A máquina de servidor remoto não existe ou está indisponível (Visual Basic)
Você está tentando se conectar a um servidor remoto que não está disponível ou não existe. Esse erro pode ser causado por sua conexão de rede sejam perdido.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Reconecte à rede e tente novamente.
