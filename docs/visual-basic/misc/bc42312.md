# <a name="xml-comment-block-cannot-be-associated-with-any-language-element-that-supports-the-application-of-xml-documentation-comments"></a>Bloco de comentário XML não pode ser associado a qualquer elemento de linguagem que ofereça suporte a aplicação de comentários de documentação XML
Bloco de comentário XML não pode ser associado a qualquer elemento de linguagem válido. O comentário XML será ignorado. Esse erro também ocorre se uma marca XML no bloco de comentário não tiver nenhum elemento de idioma correspondente.  
  
 **ID do erro:** BC42312  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Localizar e remover a marca XML externo ou fornecer o elemento de linguagem que ele descreve.
