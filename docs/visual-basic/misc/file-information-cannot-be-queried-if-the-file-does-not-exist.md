# <a name="file-information-cannot-be-queried-if-the-file-does-not-exist"></a>Não não possível consultar informações do arquivo se o arquivo não existe
Foi feita uma tentativa de consultar um <xref:System.IO.FileInfo>objeto para obter informações sobre um arquivo que não existe.</xref:System.IO.FileInfo> Geralmente isso ocorre devido a um erro ao fornecer o nome do arquivo.  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Verifique se você está fornecendo o nome de arquivo correto.
