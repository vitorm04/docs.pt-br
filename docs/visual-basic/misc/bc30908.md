# <a name="type-typename-cannot-inherit-from-a-type-nested-within-it"></a><span data-ttu-id="745c7-102">O tipo ' \<typename > ' não pode herdar de um tipo aninhado dentro dele</span><span class="sxs-lookup"><span data-stu-id="745c7-102">Type '\<typename>' cannot inherit from a type nested within it</span></span>
<span data-ttu-id="745c7-103">Uma definição de classe ou interface inclui uma [instrução Inherits](../../visual-basic/language-reference/statements/inherits-statement.md) que especifica um tipo aninhado dentro dela.</span><span class="sxs-lookup"><span data-stu-id="745c7-103">A class or interface definition includes an [Inherits Statement](../../visual-basic/language-reference/statements/inherits-statement.md) that specifies a type nested within it.</span></span>  
  
 <span data-ttu-id="745c7-104">A herança deve ser linear, não circular.</span><span class="sxs-lookup"><span data-stu-id="745c7-104">Inheritance must be linear, not circular.</span></span> <span data-ttu-id="745c7-105">Um tipo não pode herdar de um tipo que herda dele.</span><span class="sxs-lookup"><span data-stu-id="745c7-105">A type cannot inherit from a type that inherits from it.</span></span>  
  
 <span data-ttu-id="745c7-106">Uma restrição relacionada é que um tipo não pode herdar de um tipo que ainda não está definido.</span><span class="sxs-lookup"><span data-stu-id="745c7-106">A related restriction is that a type cannot inherit from a type that is not yet defined.</span></span> <span data-ttu-id="745c7-107">A herança envolve a capacidade de reutilizar membros da classe base, que por sua vez requer que esses membros sejam definidos.</span><span class="sxs-lookup"><span data-stu-id="745c7-107">Inheritance involves the ability to reuse members of the base class, which in turn requires that these members be defined.</span></span> <span data-ttu-id="745c7-108">Portanto, Visual Basic não pode compilar um código como o exemplo a seguir.</span><span class="sxs-lookup"><span data-stu-id="745c7-108">Therefore, Visual Basic cannot compile code such as the following example.</span></span>  
  
```vb  
Public Class outerClass  
    ' The following statement is INVALID because innerClass is not defined.  
    Inherits innerClass  
    Public Class innerClass  
        Public Sub doSomething()  
        End Sub  
    End Class  
End Class  
```
