# <a name="39option-explicit39-can-be-followed-only-by-39on39-or-39off39"></a>&#39;Opção Explicit&#39; pode ser seguida somente por &#39;em&#39; ou &#39;Off&#39;
`Option Explicit` as instruções devem especificar um `On` ou `Off`. Todos os outros valores são inválidos.  
  
 **ID do erro:** BC30640  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Especifique `Option Explicit On` ou `Option Explicit Off`.
