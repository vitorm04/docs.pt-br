# <a name="expected-closing-39gt39-for-xml-tag"></a>Fechamento esperado '&gt;' para a marca XML
A marca XML para um literal XML está faltando o fechamento necessários ' >' caracteres.  
  
 **ID do erro:** BC31166  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Adicionar o fechamento ' >' caracteres ao final da marca XML para o literal XML.
