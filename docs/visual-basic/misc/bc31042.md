# <a name="39sub-new39-cannot-implement-interface-members"></a>&#39; Sub novo &#39; não é possível implementar membros de interface
`Sub New`é um construtor e ele não pode implementar membros.  
  
 **ID do erro:** BC31042  
  
## <a name="to-correct-this-error"></a>Para corrigir este erro  
  
-   Remover `Implements` instruções de `Sub New` procedimentos.
