# Conex&#227;o para o tipo de biblioteca de objetos para processo remoto foi perdida
Perdeu a conexão com a biblioteca de objetos do processo remoto ou biblioteca de tipos.  
  
### Para corrigir este erro  
  
1.  Reinicie o `Application` objeto.  
  
2.  Na caixa de diálogo de erro através da qual você entrou neste tópico da Ajuda, clique em **OK** para exibir o **referências** caixa de diálogo. A referência perdida aparece com a palavra "Ausente" à sua esquerda.  
  
3.  Exclua a referência perdida.  
  
4.  No **referências** caixa de diálogo, selecione a caixa de seleção o `Application` objeto.
