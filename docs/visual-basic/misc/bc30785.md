# Especificador de par&#226;metro &#233; duplicado
Um especificador de parâmetro é aplicado mais de uma vez para o parâmetro.  
  
 **ID do erro:** BC30785  
  
### Para corrigir este erro  
  
-   Remova os especificadores duplicados.
